    旅行记录之北京

    标签： 无 分类： 未分类 创建时间：2021-04-02 08:33:07 更新时间：2025-04-28 14:37:41

    无论是我自愿也好，无论是被迫也好，2021年的清明节，我还是踏上了去北京的旅程。我还记得上一次去北京的时候，也是三天的时间，主要是参加Esri开发者大会，那时候还是在测绘院的时候，算是公费旅游了一趟。大会持续了两天，有很多的会场和课程，不能说自己全部都把心思放到了听课上，当然也没有白去，还是听了几节课的。那个时候应该是夏天吧，天气还是挺热的，我还依稀的能记得，我去了北京天安门，进了午门，但是没有进故宫。我去了后海，酒吧一条街，吃了点不知道是什么的小吃。还走过了王府井，就这么一个人在熙熙攘攘的人群中，来回的踱步。去了北京师范大学恭王府的那个校区，真的是古老的建筑，有高高的城墙，有窄窄的窗户，连门都是又高又瘦的，让人以为那里面会有一个不一样的乾坤。最后一天的开会，是去听了相声，虽然不怎么搞笑，但是也还算可以了吧，算是一个小剧场。剩下的，就是和朋友吃了一顿饭。
    这一次去，再次去北京，要待三天，不知道该怎样的安排啊。从开始谋划到最终决定买票，总共用了不到两个小时，真可谓是一场说走就走的旅程啊。作为一个死宅(本人不是很肥)，天知道我这么费心费力的去一次北京，要下多大的决心，要耗费多少的能量啊。

    1.买票

    关于买票，其实我只是临时起意要去北京的，买了去程的机票“2021年04月03日08:30-10:04 厦航 MF1994(实际承运南航CZ8854) 杭州(萧山国际机场T3)-北京(大兴国际机场)”，返程的机票是“首航 JD5357 北京(大兴国际机场)-杭州(萧山国际机场T3) 2021年04月05日23:15-2021年04月06日01:15”，看看这个时间，就知道回来的时候会很累，那个时候已经没有了地铁，还要打车，不知道能不能打到车。我刚开始是携程上看的航班和票价，这个日子，往返的票价总共是1494元，后来我又去了智行上，查看了相同的航班和票价，发现最后加上燃油费，往返的票价只要1446元，便宜了48元，于是也不是果断的，纠结了很久最后还是在智行上买了飞机票。携程上虽然贵，但是发票开的很及时，可以开电子发票，即便是两个人，还是可以开到一张发票上，这样对于报销来说，还是很合算的。智行上，发票就比较难办了，要么就是自己去机场拿，要么就是花20块钱让他们给寄纸质的发票，所以还是挺麻烦的，有一次我打电话去问智行的工作人员，说开发票还要很久才能寄到，反正就是挺麻烦的。

    2.行程安排

    没有办法，都是临时拼凑的行程，该去哪，该怎么走，都要临时的安排。所以啊，就在出现的前一天，我们开始疯狂的补习功课。到底该怎么走。最后也还是做了半吊子的规划，即便是这样，也没有按照预定的计划走，预定的路线走，以及不可抗力(主要是人多)等原因，最终，很多的地方都没有去到，而很多的地方，花费的时间又相当的长。规划如下：

    D1 天安门广场(1.5小时) → 前门大街(1.5小时) → 北京全聚德(前门店)(1小时) → 故宫(3小时) → 景山公园(1小时)
    D2 天坛(2.5小时) → 雍和宫(2小时) → 孔庙和国子监博物馆(1.5小时) → 簋街(2小时)
    D3 恭王府(2.5小时) → 什刹海(3小时) → 南锣鼓巷(机动选择，1.5小时) → 什刹海酒吧街(2小时)
    D4圆明园(3小时) → 清华大学(2小时) → 鸟巢(1小时) → 水立方(1小时)
    D5颐和园(4小时) → 北京大学(2小时)
    D6八达岭长城(3-4小时) → 明十三陵(2小时)

    六天的行程，就包括了全部的北京的景点，每天都安排的满满的，又不至于很累。

    想法都是挺好的，可是最后的结局总是让人史料不及。第一件事，就是北京，最重要的历史性景观故宫没去成。等我们从决定去北京，到买票出发，然后再去搜攻略，只用了半天不到的时间，所以临时去看故宫的门票，竟然已经全部售罄了，4月3号，4月4号一直到4月5号的票，都没有了，只有4月6号的票还有，那个时候，我已经在上班了。第二件意外之中却又在情理之内的事件，就是人真是超级超级超级超级的多，所以如果有条件，一定不要在节假日的时候，去一些热门的顶点，如果非得要去，一定会让人怀疑人生。

    1.出发

    4月3号早上六点中起床，天气阴沉沉的。简单的洗洗刷刷，带上身份证、雨伞，换洗衣服，6点20就出发前往萧山国际机场，坐地铁一号线，萧山国际机场，已经不是第一次来了，熟悉而陌生。打印好登机牌和报销单，就去登机口等着了，也没等多长时间，就开始检票了。原定于8点30分起飞的航班，因为航空管制，最后不得不推迟一个小时起飞，原本就紧张的形成，又在飞机上浪费了一个小时，从杭州到北京大兴，本来之需要1小时40分钟的飞机，现在花费了整整两个小时40分钟。

    2.北京大兴国际机场

    晚了一个小时，到达北京大兴的时候，已经中午十二点了，坐摆渡车到航站楼，然后出门，因为去北京城区有地铁，一个人花费35块钱坐两站，就可以从大兴到草桥，然后再转10号线，去游览北京城的其他地方。

    3.圆明园

    到北京很晚，第一站就直接从大兴机场，坐地铁去了圆明园。门票也不贵，只要25块钱。圆明园的历史我就不说了，网上一大堆，我只能说，现在什么东西都没有了，只剩下了一个公园，和其他的公园一样，就是花花草草，树木和湖泊。进院之后，恰好遇到了北京工业大学的学生组织清明活动，于是就跟着他们以及导游，从大门口一直走到三一八公墓那里，路上导游会偶尔将一些历史故事，讲了园子的历史，英法联军、八国联军，以及建国之后的无政府管辖时出现的人为破坏，很多老百姓都把园子里的石头拿回家盖了房子，或者把树木砍伐等等，一直到2008年，最后一户人家才从圆明园迁走。每当我听到这些历史的时候，我总有一种莫名的伤感，听到英法联军火烧圆明园，抢走了数以万计的珍贵文物，我都觉得很可惜，不完全同统计，被抢的文物多大115万件，这是多么巨大的一笔财富啊，说可以买一个小国，应该不成问题。历史，终究时历史，后人也只能为戒，珍爱和平，不断强大，才能不让历史重演。
    大学生去公墓了，我们就启程去了书本上那张著名的圆明园图片，那个破乱的石柱那里。导游说在园子的东北角上，只要一直朝着东北角走就可以了。以前不知道，现在才知道那个柱子叫大水法，位于西洋楼景区，是乾隆时期建造的，以前的辉煌和历史地位我也不说了，有机会可以去横店的圆明园看一看，复原后的圆明园是不是真的有那么气魄。我们一路往西北走，在地图上，我以为过了那个坡就可以到西洋楼景区，上去之后才发现，被铁网围起来来，无奈之能回到位于西边的入口处重新入园了。因为我们买的是大门票+西洋楼景区+全景模型，所以排个队就可以直接进入了。
    进入景区，第一个叫什么谐奇趣，没文化的我，只能看到一片大理石废墟，还有一个迷宫，再往东走，还有线法桥、蓄水楼、养雀笼、黄花阵（万花阵），反正都是破烂不堪了，就是在这破烂不堪中，可以想见当时全胜时期的巍峨壮观，让这伙野蛮的英国人和法国人给毁了，真的是可恶啊。我们就一路顺着景点往东走，一路拍照，出景区的时候，已经差不多五点了。那个圆明园全景也看不见了，走到湖边的时候，夕阳照在湖面上，湖面映出群山，真有西湖上雷锋夕照的样子，很美。
    从圆明园东北门出来，天已经差不多黑了，旁边的清华大学，北京大学，也没有机会继续浏览了，于是坐车去了后海。

    4.南锣鼓巷

    我们先打车到地铁站，然后15号线转8号线，到什刹海站下车，跟着导航，去到了一个叫“姚记炒肝”的小店，为了去吃北京当地的小吃。人还真多，里面的桌子已经坐满了人，还有很多的人在排队买单，拿菜。好在东西都是已经做好的，排队也很快的，排了20多分钟，就到了。点了豆汁、焦圈、小炒肝、卤煮，还有包子，好不容易找了个位置坐下，开始吃这么东西。包子还可以都，没啥特别都，就是一个包子，小炒肝，里面就是几个大肠和猪肝还有晶莹剔透都那种冻冻块一样都东西，卤煮里面也是大肠和猪肝以及其他都下水和火烧；炸灌肠就是一片一片都东西，说不上来。最奇特都就是那个豆汁了，和我们平常吃都豆汁不一样，这是一种豆渣弄都有点酸酸臭臭都东西，反正我自己是吃不惯这个东西，还有一个类似于麻花一样都焦圈。
    吃完了当地都小吃，出门左转，走不远到了一个胡同，这里就是南锣鼓巷，其实也就是在胡同里面都一个小吃街，和其他都小吃街没啥区别。胡同都两边分布着各种个样都小店，还有卖鼻烟壶都，卖猪肚等等，我们没有走完，就在中间的一个小胡同里拐出来了，去往了后海方向。

    5.后海

    后海酒吧一条街，真的是酒吧一条街，在后海和前海交界处的湖的两岸，分布着大大小小的酒吧有十几个，每一个酒吧里面都有人助唱，灯光昏暗，气氛暧昧，门口有些人在拉客，“几位？进来坐坐吧。到家了，不进来坐坐吗？”，这就是酒吧的氛围吧。从南锣鼓巷穿过胡同的小路，就可以直直走到后海和前海了，这里顺带着有一个小吃街，叫烟斜袋街，和其他的小吃街一样，这里也是有各种的炸串、臭豆腐、卖小饰品的，不一而足。在酒吧一条街上逛了逛，走了走，因为都是透明玻璃的，所以里面唱歌什么的都是可以听的见的，就是没有凳子坐罢了。走了一圈，听了几个店铺的助唱，吹了吹后海的湖风，已经快十一点了。
    最后纠结了很久，还是找了一家靠近桥的酒吧，叫麒麟酒吧。里面有三个助唱，两男一女。服务小哥把菜单拿上来，我真是惊呆了，怪不得会有助唱，一杯咖啡就要60块钱，最低的饮品就是60块了，啤酒更贵，有一千多的，两千多的，还有三千多的，鸡尾酒一杯也要100块钱，起的名字倒是挺精致的，什么桃花运、原谅、新生…名字花里胡哨的，反正就是很高大上就对了，我们点了一杯新生和原谅，一杯一百块。饮料上来了，就是一个很高的杯子，用三块冰块撑起来，里面放了饮料，一个吸管，一朵花，杯子里最多的就是冰块了，占了5分之4有的。原谅，这一款真是难喝的不行，没有一点酒味，还酸的不行，那杯新生倒是挺温和的，说不上什么味，就是比起来还能下嘴吧。
    一遍喝酒，一遍听歌，我其实不喜欢这个氛围，唱的歌我也听不懂，不是听不懂，我觉得没有啥好不好的吧，反正有那个调调，发现其他人还听的津津有味，我也很不理解，我都快睡着了。就这样一直在里面待到了12点，出来的时候，酒吧街上的人明显变的少了，因为地铁已经不通了，于是就打车去了酒店。

    角门西速八酒店

    在后海、铜锣鼓巷附近，一搜就是要300多的，为了省钱，只能往三环、四环、五环的方向走了，于是就到了三环之外的角门西找了一家速八酒店，价格237。即便价格已经很高了，但是里面的设施确实非常的普通，房间里甚至没有了一次性拖鞋，还是我自己下楼去问服务台要的，只剩下一双了。房间的枕头上，还有一滩黄色的印记，估计是洗不掉了，却又不舍得换，窗户其实也很小，不过已经很晚了，凑活着睡一晚吧。

    6.八达岭长城

    早上起来已经差不多七点半了，出门到了一个街边的小店，吃了点早餐。喝了一碗豆浆，吃了一个咸的糖饼，一个甜的糖饼，加上一个火烧夹肉，为了中午不饿，于是分别又买了一份。
    吃完早饭，坐地铁从角门西到积水潭，去做877公交车到长城。人是真的多，多的让人怀疑人生。从积水潭地铁口出来，顺着人流，走了很长的一段路，到了德胜门下，这里已经开始排很长的队了。根据现场工作人员的指示，我们就顺着路往桥下走，走到桥下，有走了300多米，才到达了队伍的末尾。要说排队也快，毕竟车很多，来一量走一辆，所以队伍虽然长，但是走的也还算快。本来想上个厕所来着，可是最近的厕所已经坏掉了，离开队伍走远，回来又不知道排到什么时候了，索性就憋着好了。
    上了车，刷卡买票，售票员说，全程大约70分钟，我一听，还好的啊，也就一个多小时。都是理论上，理论和显示还是差的有点远。我们9点多坐上车，走了高速，我直接就睡觉了，等我迷迷糊糊的醒来的时候，一看已经10点15了，可是还是在路上，车子走一小段，停五分钟，走一小段，停五分钟，打开高德地图一看，才刚刚走了一半，前面一片红，堵车堵的人心理发慌，都要骂人了，坐在旁边的小伙子，一个劲的骂人。心情烦躁的时候，还是睡觉好了。堵啊，堵啊，挪啊挪，十一点半才过了居庸关，过了居庸关，路就顺畅多了，车子的速度也上来了，路上没多少车的感觉，真是爽极了。
    差不多十二点的样子，终于是开到了。沿着公交车的路线往上走，车子倒是挺多的，人也是挺多的。因为到了中午了吗，上去的人还是络绎不绝。进长城的检票口，有免费存包的地方，于是我先把身上的包存了一下，这样可以轻轻松松的爬山。山上种满了那种白色的花，我不知道是什么花，反正就是白色的就对了，然后满身都是，不是那种铺满的满，而是会露出裸漏的山体的满。因为疫情的原因，只开了北一到杯八楼，还有南一到南八楼,我们选择了爬北楼，这里据说是八达岭长城最好的风景，当然人也超级多。
    不爬不知道，一爬真是吓一跳啊，从北一到北四，人虽然多，但是还能走的动，到了北四开始，人就挤在一起了，人头挨着人头，人脚挨着人脚，就是那种坐公交车的感觉，走一步，停五分钟，走一步，停五分钟，太阳又很大，晒的都快冒烟了。

    7.甘家口大鸭梨餐厅

    搜了北京的一些烤鸭牌子，最著名的可能就是全聚德吧，以前在杭州的一个全聚德餐厅吃过，也没有什么特别的，这次来到北京，索性就吃点不一样的吧。全聚德的烤鸭是最贵的，大鸭梨的还好，半只鸭，只要九十八，当然，你像什么蘸料啦，面皮啊，都要自己重新买，要吃一次鸭，半只就要110块钱，还点了其他的菜，一个茄子四季豆，还有一个价值30块钱的蔬菜沙拉，你知道吗，就是几个烤肉中送的生菜叶子，半了一盘，就要30块钱，真是让我大跌眼镜。

    8.前门大街观旗宾馆

    花费：265
    吃完烤鸭，于是就准备去天安门看看夜景，先做公交然后转地铁坐到了天安门西站，出来的时候，还是曾经的道路，唯一不同的就是块走到天安门的时候，过安检的地方就被拦下来了，说是已经清场了，不能再进了，我记得以前不是这样的，上一次我来的时候已经查不到十一点了吧，难道我记错了？这一次才刚刚到十点就清场了啊。没办法，只能会宾馆了，绕过人民大会堂，走人民大会堂西路，就可以到前门大街了，我定的宾馆，就在前门大街上。
    本以为在天安门前的宾馆会很贵，所以第一天就住在了外围，角西门，加上打车35，住宿231，也是差不多花了266。可是晚上临时定的时候发现了一个天安门的观旗宾馆，就像名字一样，真的就在天安门广场的前面大街上，过条马路就到了天安门广场，携程上定的时候，只需要265块，本来定的时候，还担心会非常的差，等到吃饭完之后，去到宾馆，竟然意外的干净啊，除了隔音效果不好，其他都非常的不错。隔着马路就可以看到天安门，看到天安门广场。回去的时候11点。我还不太困，就出宾馆了。

    9.天安门升旗

    早上定了四点十分的闹钟，闹钟响了，墨迹了一会，差不多4点30出门，路上已经有人陆续的开始往天安门赶了。我们紧赶慢赶，超过了一群又一群的人，到达广场的东南角的时候，人已经排了很多了。刚开始以为要身份证，可是我出来的太匆忙了，就忘记带了，好在最后没有身份证，可以报身份证号码进行安检。过了第一道栅栏，然后跑了一段，继续排队过安检，从一个地下安检口出来，就可以到广场上了，那个时候，人已经很多了，大家都像疯了一样的，往国旗的方向狂奔，能超过一个是一个吧。等我到观看国旗的地方，前面已经围了六七层的人了，加上隔着旗杆很远，基本上啥都看不清了，这个时候，看看手机时间，也才刚五点钟，要一直等到5点52分，太阳出来才可以。除了天安门的正面有人以为，在天安门金水桥的两侧也有很多的人，除了人还是人。
    就这样站着，站着，站了四十几分钟，穿过层层的人头，略过一个又一个的手机，突然人群中出现了骚动，很多人说，来了，来了，开始了，有小孩的，就把本来已经趴在地上的小孩子抬起来，架在脖子上，这下我是彻底的被挡住看不见了。什么也看不见了，人家唱国歌，我也就跟着唱，什么时候唱完的，什么时候结束的，都不知道。等了一个小时，起了一个大早，啥都没看见，现在想想，我真是看了个寂寞，就是去排了个队。
    升旗很快，几分钟之后就结束了，即便是结束了，安保人员把围栏打开，人群还是蜂拥进去了，就是为了离国旗更近一点。许久许久，还是有很多人在那里等着，等着，不知道在期待些什么。因为我们住的比较近，加上气温还是有些低的，于是乎待了一会，就从天安门广场的西侧回到了宾馆，又睡了一个回笼觉。

    10.毛主席纪念堂

    等我再次醒来的时候，已经八点了，墨迹墨迹，收拾收拾，出门已经九点了。去了前面的前门大栅栏社区路边的一个叫护国寺小吃的地方，买了点早点，然后去吃了什么河肉饼，叫啥我也忘记了，反正就是卖肉饼的地方。
    吃完饭，我们就从天安门广场都左侧都那个安检通道，也就是我们早上看完升国旗之后出来都地方，进去天安门广场，人还是很多都，三个通道，还是排了很长都队伍。
    进去都第一站，其实就是那个巨大都前门楼子了，在前门楼子拍了几张照片，绕过毛主席纪念堂，顺便拿出手机看看了看，发现还能预约毛主席纪念堂都票，我们就约了，然后去对面都国家博物馆免费存包处，存了个包，因为里面不让带包，也不让带相机，所以只能把包存着。然后就排着队伍进去了。纪念堂非常都庄严，因为不让拍照，所以也只能回忆了。进门就是一个巨大都毛主席雕像，坐在沙发上都白色雕像，前面放了很多黄色都花，因为排队进纪念堂的都时候，有卖花的，三块钱一朵，三朵起卖，于是我们两个人买了三朵，进去献给了毛主席。有些人，在雕像前双手合十，庄严的鞠了鞠躬，我不知道他们在想些什么，我在毛主席像跟前，就像我在其他的地方见到的毛主席像一样，我只是静静的伫立着，伫立着，凝视着，仰望着，这个缔造了中华人民共和国这一个伟大国家，又在晚年犯过重大错误的人，他是伟大的，不朽的。如果他还在世，或许我这辈子都不会遇见他，见他一面。
    进门之后，我们朝右手边走，绕过汉白玉雕塑都毛主席像和一面墙，就可以来到后堂了，后堂就是放着毛主席的遗体，有很多的部队人员在左右站立。顺着步道缓缓走过玻璃棺，看不清楚毛主席的全脸，只能看到半个侧面，给我的感觉还是和电视里、照片上以及影像资料中见到的不一样，历经四十多年的药水浸泡，可能早就变样了吧。我不知道为什么会把他的遗体放到这里，为什么不能尘归尘，土归土，让这位伟大的共产主义战士卸下自己的全部，安详的离开这个世界，这也不是我一个小小的平民能够考虑的事情。我也直想感谢这位伟人，这个伟大的老人，并向着他指明的道路前进。
    书到用时方恨少，总想感慨些什么，最后还是随着人群走出了大堂。

    11.天安门

    从纪念堂的南门出来，还是那个前门楼子，再向北绕过毛主席纪念堂，就是天安门广场的人民英雄纪念碑。我们排队取了东西，就去看了天安门。天安门我也不是第一次来了，上一次来的时候，是晚上和下午，也从金水桥过了天安门。天安门前就是长安大街，天安门城楼上挂着毛主席的巨型雕像。想要穿过天安门，就要走金水桥，桥上还是有便装的部队人员在组织，让人们快速通过，不要逗留。天安门的两个侧面是只能进不能出的，中间的一个门是不开放的，上面挂着高6米，宽4.6米，重达1.5吨的毛泽东主席画像。就像在电视里看的一样，天安门就是红砖绿瓦的一个大大的门。
    天安门进去是一个类似于广场的一个地方，前面就是端门。我们一般就是走正门进入。

    12.故宫

    从端门进入，就是午门，也就到了故宫博物院的入口，因为我们没有订到故宫的门票以及国家博物馆的门票，所以没有办法进到里面去参观。如果要赶节假日去北京，记得提前十天预定故宫和国家博物馆的门票，现在也没有了线下售票的说法，订不了也就意味着进不去了。进不了故宫博物院，我们只能在午门前的广场上休息了一会，然后走左侧的中山公园出口出去了。下一站就是故宫的后门。

    13.景山公园

    从故宫出来，过了护城河，然后到北长街上，一直向北走，绕过故宫，到底故宫的正后门，对面就是景山公园了。我一直很好奇，北长街上会住着什么样的人呢？这么个核心地位。走到一个小院子的时候，里面有人开门，出来的是穿着军服的人，正在盘查一量想要进门的车辆，我就知道有部队的人就不是简单的人物住在里面的。
    景山公园有一颗歪脖子树的位置，就是当年崇祯皇帝自缢的地方，当然原树已经被砍了，这棵树是后来从别的地方移过来的。景山其实一点也不高，上面有两层，较低的那层建筑叫啥我忘记了，最高的那层上面的对面就是故宫，所以来到景山，就是为了在这个平台上看故宫的全景。

    14.北海公园

    从景山公园的西门出来，对面过一条马路，往里走就是北海公园。北海公园有什么好的呢？就是“让我们荡起双桨，小船儿推开波浪，湖面倒映着美丽的白塔和绿树红墙”，就是说的这里了。门票也不贵，只要十块钱，可以爬到山上，看那一个小小的白塔。据传很久了，这座塔，不过要是想上去，还是要花10块钱。这个塔和我们常见的塔不一样，是藏传佛教的塔。
    北海公园本来就小，我们也没有留恋，就顺着湖的一边，往北海公园的北门走，准备去恭王府。

    15.恭王府

    什么叫有钱人家，这就是有钱人家。从北海公园出来，然后顺着高德地图指示，走个一二十分钟，就可以到了恭王府。恭王府，最早为和珅的家，后来赐给了恭亲王奕訢，所以叫恭王府。里面有多大，有多豪华，已经不能想象了，反正园子很大，很多的东西也都是后来修缮的结果了，在那样一个动乱的年代，加上后来的文化大革命，等等，或许在古老的物件，都不会剩下什么吧。里面的花倒是还是那么的旺盛，一整个的花树，风一吹，特别的适合拍照。

    16.聚宝源后海店

    这个店也是大众点评上搜的，据说也是老北京火锅，是一个连锁店，应该和东来顺差不多。从恭王府出来，顺着北京古老而现代的胡同，就往后海那里走，因为和朋友约好了吃一顿北京的老火锅，于是就选择了这家店，离地铁什刹海站挺进的。老北京火锅，其实也就是火锅，一个高高的有烟囱的锅，加上清水，一半加清水，一半加辣椒，就成了鸳鸯锅了。

    17.返程

    吃完饭，八点半开始从后海往大兴机场赶，8号线、4号线、16号线、大兴机场线，花了差不多1个半小时，到达大兴机场的时候已经十点了，机场很大，从地铁下来，没走多远就是在线值机，值机的时候，不用打印登机牌，刷身份证就可以了。但是这个值机的机器不太好的地方，就是它不能直接识别身份证，要先选择哪个航空公司，然后才能刷身份证值机，而不是直接将身份证放上去，直接显示你的行程。值完机，就直接去了安检，我本来还想着打印发票，最后也没有找到，安检进去，也没见到有航空公司的服务台，就是问了问询台的工作人员，说已经很晚了，没有地方打印发票了。没办法，只能放弃了。
    机场是真的大，我就一直走，一直走，一直去找D68登机口，在值机时发送的短信，时22:35开始登记，让我变的更加的慌了，已经到了10点30了，还是没走到登记口。经过小跑加滑行，好在还是提前赶上了，飞机23:15分起飞，好在没有晚点，上了飞机就开始睡觉，真的是累的不行了，虽然也是睡不着，姿势非常的不舒服，但是啊，耐不住一天的折腾，还是迷迷糊糊的睡着了，等我醒来的时候，已经快要着陆了。
    回到杭州差不多12点半左右，没有了顺丰车，没有了拼车，更没有了地铁，只能打车了。我不喜欢在还没有出站的时候，就开始打车，于是还是走到了国内出发那层楼上，打了车，车子不好打，出站的人多，车少，还不让停很长时间，所以整个出发层，显的很乱，车子来了又走了，走了又来了。好不容易从排队的30号人中，排到了第一，但是一连有五六辆车，都是接了单之后，又取消了，等了20分钟，才最终等了一个司机没有取消订单。上车接着睡，到家已经两点了，从机场回家，总共花了118块钱，加上20块钱的过路费。连澡都没洗，就睡觉了，实在太累了。
    北京之行，就这样画上了句号。

    3.总结

    三天下来，其实真都挺累的，走了很多的地方，大部分的时间都是在排队，特别是第二天去八达岭长城，就是非常的离谱，高速路上就堵了两个小时，上长城又堵了一个小时，一整天都耗在了长城上。因为出门比较匆忙，其实还是没有约上故宫和国家博物馆的门票，其实挺可惜的。去了一趟北京，收获是什么呢？收获了风景，和一些记忆，这些记忆会尘封在某一个角落中，不知道什么时候才能再次唤醒。就像路上看到的美女一样，只是当时觉得很美，可是等过了一天，两天，就再也不会记得有这么一档子事情了。美好的东西，留在回忆了，人生很短，明天在继续，回忆会不断的增加，很多的东西，根本来不及回忆，就已经被新的回忆取代了。

    isNaN 和 Number.isNaN 函数的区别？

    1.函数 isNaN 接收参数后，会尝试将这个参数转换为数值，任何不能被转换为数值的的值都会返回 true，因此非数字值传入也会返回 true ，会影响 NaN 的判断。

    2.函数 Number.isNaN 会首先判断传入参数是否为数字，如果是数字再继续判断是否为 NaN ，不会进行数据类型的转换，这种方法对于 NaN 的判断更为准确。

    一个直接的解决方法就是设置一个误差范围，通常称为“机器精度”。对JavaScript来说，这个值通常为2-52，在ES6中，提供了Number.EPSILON属性，而它的值就是2-52，只要判断0.1+0.2-0.3是否小于Number.EPSILON，如果小于，就可以判断为0.1+0.2 ===0.3。

    == 操作符的强制类型转换规则？

    1.首先会判断两者类型是否**相同，**相同的话就比较两者的大小。

    2.类型不相同的话，就会进行类型转换。

    3.会先判断是否在对比 null 和 undefined，是的话就会返回 true。

    4.判断两者类型是否为 string 和 number，是的话就会将字符串转换为 number。

    5.判断其中一方是否为 boolean，是的话就会把 boolean 转为 number 再进行判断。

    6.判断其中一方是否为 object 且另一方为 string、number 或者 symbol，是的话就会把 object 转为原始类型再进行判断。
