    人生经验之三胎

    标签： 无 分类： 未分类 创建时间：2023-04-19 07:59:29 更新时间：2025-04-28 14:37:41

    前言

    这篇文章是从一个公众号 “力哥理财” 里面摘录的，原题为《 国家不是放弃躺平族，而是舍不得出钱 》，因为写的太现实，被封禁了，我这里作为一个简单的记录，不发表自己丝毫的态度。

    原文

    昨天开放三孩消息一出炉，网上就炸锅了，各种段子满天飞。我司20多号人，就我有孩子，其他全丁克或单身汪，一个娃木有，也在工作群里讨论热火朝天，吐槽扯淡打哈哈的内容，昨天大家都看够了，今天力哥说点正经的。

    从传统社会地少人多，到后现代社会老龄化导致劳动力不足，国家的人口政策从计划（限制）生育转为鼓励生育，是一件既符合逻辑，也符合发达国家经验的事，没啥稀奇。问题在于，转变节奏，如何把握。毕竟中国是一个14亿人口大国，地大物博，各地情况差异很大。中国人口问题核心，是总量还太多，但结构已失衡，想要快速改善结构，就有大幅增加总量的风险。所以中央出台大幅转向政策，必须慎之又慎。但不管如何审慎，大势所趋，该来的总会来，无非早一年转还是晚一年转，越早越主动，越晚越被动。

    生育政策转向，一般三步走：

    第一步，适度放开计划生育；

    第二步，完全放开生育（取消计划生育）；

    第三步，口惠实至，鼓励生育。

    2014年，开放单独二孩，2016年，全面开放二孩，第一步就走完了。第二步，理应在2018-2020年完成。推迟到2021年，主要是上头看到七普数据。虽然数据公布意外推迟，公报内容也有明显“政治正确”味道，但从开放三孩速度看，上头明显感觉到压力，问题紧迫，必须马上行动。以上，全在意料之中，没啥稀奇。稀奇的是，这次中央出台的决定是：进一步优化生育政策，实施一对夫妻可以生育三个子女政策及配套支持措施。怪了，难道不应该是直接取消计划生育吗？

    开放二孩，边际效应还比较明显。只有一娃和有两娃，无论对孩子健康成长还是分散父母养老风险，作用都立竿见影。但开放三孩和四孩五孩，差异微乎其微。有能力也有意愿生三孩的，要么完全不差钱，要么很差钱，但完全不在意孩子教育培养。前者极少数，后者也是少数，今天大部分80-90-00后，都受过比较好的教育，多少有点见识，也没那么富，不至于那么想不开，拼命生三孩。这次只开放三孩，过两年是不是还要逐步开放四孩、五孩呢？那就很扯了。因为边际效应几乎忽略不计，实际没啥用。所以这次新政的关键词，我觉得是“配套支持措施”。这其实是第三步，鼓励生育。意味着，我们一边依然施行计划生育，一边已在鼓励生育。要限制生育，就不该鼓励；要鼓励生育，就不该再限制。乍一看，矛盾啊。有人说，因为计生委那么多人不能一下子都下岗，得给人家一口饭吃。简单啊，直接把计生委改命促生委，把工作内容从限制生育改成鼓励生育就行了。所以我觉得问题的核心在于：“国家想让年轻人多生，又不想掏太多真金白银鼓励，就先从比例最低的三孩开始鼓励，控制开支。”

    老龄化、延迟退休、鼓励生育、教育内卷、房价调控等等问题，归根到底，就是一个利益分配问题。政府与人民，人民与人民，中央政府与地方政府之间的利益博弈。政府希望多生，但年轻人肯定说：你别光嘴上BB，要切实减轻我们生育负担啊！教育和房子这两座大山不减负，生个球？“配套支持措施”这6个字，意味着今后肯定有实惠。但先从三孩开始，你要拿到国家实惠，得先把前两娃生出来，而国家没有实惠。这样国家成本可控，只用一个红包，换来3个娃，投入产出比更高。

    有人说，这次政策是国家层面的内卷：不鼓励增量（不婚不育族），只压榨存量（已生了2个的继续生），国家已放弃平躺族。错。国家只是觉得给点小恩小惠，就能有效促进二孩父母再生一个；但要鼓励不婚不育平躺族生一个，得花上几倍代价。不划算。

    从改善人口结构看，鼓励二孩父母再生一个，也比鼓励平躺族生一孩更有效率。愿意生三孩的父母，要么不差钱，能给孩子最好的教育，大概率以后培养出的也是精英，国之栋梁，国家最希望富人多生孩子。另一类是没见识的底层穷人，自己吃了一辈子苦，还被传统思想束缚，把生娃当成劳动力投资。但对孩子来说，家里一穷二白，大学学费还得贷款，下有弟妹要供养，上有父母要养老，自己还要成家，怎么可能平躺，必须当奋斗逼，不得不拼命向上，持续奋斗，我爱996，更爱007。国家同样需要大量底层奋斗逼，不然谁去盖房子，谁去送外卖，谁给养老院老人把屎把尿？这么一分析，你是不是豁然开朗？

    从这次中央释放的鼓励生育方向（由多到少、由易到难），我们大体能预测未来的政策走向：

    1、1-2年内，落实三孩鼓励政策；

    2、2-4年内，再放开四孩，或全面放开生育，三孩鼓励政策同样适用于四孩或更多；

    3、4-8年内，落实二孩鼓励政策；

    4、2030年后，落实一孩鼓励政策。

    所以我奉劝像小贝这样还没结婚就想着薅国家羊毛的小伙伴，洗洗睡吧。等国家出台一孩鼓励政策你再生娃，早错过生育年龄了。对我们投资理财，新政有啥赚钱机会不？还是4个字：洗洗睡吧。实际效果会非常有限，股票也只是一波流，绝对天坑（参见二孩开放后的走势）。与其买“三孩概念股”，比如奶粉、母婴、幼教等，还不如买吃喝玩乐的股票。不管上头怎么想，怎么做，越来越多中国青年，会和今天日本/台湾青年一样，选择不婚不育，每天吃喝玩乐，娱乐至死。

    至于学区房，教育内卷大环境不改变，也不管上头怎么做，都不太可能在短时间内彻底凉凉。但那么贵的学区房，你买得起吗？

    isNaN 和 Number.isNaN 函数的区别？

    1.函数 isNaN 接收参数后，会尝试将这个参数转换为数值，任何不能被转换为数值的的值都会返回 true，因此非数字值传入也会返回 true ，会影响 NaN 的判断。

    2.函数 Number.isNaN 会首先判断传入参数是否为数字，如果是数字再继续判断是否为 NaN ，不会进行数据类型的转换，这种方法对于 NaN 的判断更为准确。

    为什么0.1+0.2 ! == 0.3，如何让其相等?

    一个直接的解决方法就是设置一个误差范围，通常称为“机器精度”。对JavaScript来说，这个值通常为2-52，在ES6中，提供了Number.EPSILON属性，而它的值就是2-52，只要判断0.1+0.2-0.3是否小于Number.EPSILON，如果小于，就可以判断为0.1+0.2 ===0.3。

    == 操作符的强制类型转换规则？

    1.首先会判断两者类型是否**相同，**相同的话就比较两者的大小。

    2.类型不相同的话，就会进行类型转换。

    3.会先判断是否在对比 null 和 undefined，是的话就会返回 true。

    4.判断两者类型是否为 string 和 number，是的话就会将字符串转换为 number。

    5.判断其中一方是否为 boolean，是的话就会把 boolean 转为 number 再进行判断。

    6.判断其中一方是否为 object 且另一方为 string、number 或者 symbol，是的话就会把 object 转为原始类型再进行判断。
