    ArcPro安装与破解

    标签： 无 分类： 未分类 创建时间：2022-04-19 15:04:32 更新时间：2025-04-28 14:37:39

    1.前言

    刚开始我没有破解成功ArcGIS Pro2.8，后来经过了很长时间，到了Pro3.0.1成功进行了破解。

    1.准备账号

    登陆或者注册 最好是重新注册一个账号，重新注册的账号，才有23的适用许可，才能登陆ArcGIS Pro，登陆之后领取试用许可。

    参考文章:
    1.ArcGIS Pro 2.8破解版下载(带无限试用补丁) 中文离线版 这里有三种方法进行破解
    2.ArcGIS Trial: Getting Started with ArcGIS Pro & Extensions 如何进行授权配置的视屏

    2.配置许可

    在打开许可页面，在ArcGIS Pro许可处，点击管理。

    将许可授予某一个账号，我这里只有一个账号，直接就打开就可以了。

    3.试用成功

    这样配置了试用许可之后，就有了23天的试用期，其他的模块的许可，也是这么设置就好了。

    4.破解

    我这里提供了两种版本的破解，ArcPro3.0.1 的破解成功了，ArcPro2.8的破解不成功。

    1.ArcPro3.0.1

    这里提供了一个下载地址 提取码：tt6k，我没有尝试，自行尝试吧。我用了另外一个下载地址(本来想放到阿里云盘上的，结果阿里云盘不支持分享iso文件，真是够扯淡的)。 安装步骤：

    (1) 安装 用这里安装步骤进行安装就可以了,还可以通过 公众号获取破解补丁。

    (2) 替换AfCord.dll。

    (3) 删除注册表里面的内容:按键盘上Windows+R组合键打开运行框。输入 regedit 然后点击确认键，弹出注册表编辑器。按照以下路径删除程序许可证的文件夹（如果存在）：HKEY_CURRENT_USER\Software\ESRI\ArcGISPro\Licensing 和 HKEY_LOCAL_MACHINE\SOFTWARE\ESRI\ArcGISPro\Licensing,发现不用登录就可以使用ArcPro了。

    参考文章:
    1.ArcGIS Pro 3.0“学习版”可以下载了 由 i-ARer 破解，Daniel Lau 打包
    2.ArcGIS Pro 3.0.1中文破解版下载安装教程 我使用了这个进行了下载，结果下载错了，花了一天的时间，结果下载错了，应该下载学习版的。
    3.ESRI ArcGIS Pro 3.0.2破解版 没有提供下载地址

    2.ArcPro2.0

    下载安装就不说了，主要就是破解，破解文件中主要就是 AfCore_libFNP.dll文件覆盖 和 lic.reg文件 文件的双击写入注册表。我找到了一个安装步骤，如下操作，结果还是失败了，是因为我以前安装过ArcGIS Pro，后来下载之后重新安装了，所以无法激活了吗？

    (1) 安装0.ArcGISPro_28_zh_CN_177705.exe，然后卸载，保留汉化补丁(这个操作有点不好，建议直接上英文版，英文实在不行再走这一步)
    (2) 安装 ArcGISPro_28_177688.exe 和 ArcGIS_Pro_283_177646.msp
    (3) 复制 “AfCore_libFNP.dll” “AfCore_Licensing.dll” 到安装目录下替换(默认位置：C:\Program Files\ArcGIS\Pro\bin)
    (4) 运行 lic.reg
    (5) 复制 “ESRI_Licensing” 文件夹到： C:\Users{你的用户名}\AppData\Local\

    后来我又从网上(参考文章5)下载了别人的 license 文件夹，替换了自己的 C:\Users{你的用户名}\AppData\Local\ 文件夹，虽然不需要登陆了，再进去的时候，页面上也没有登陆

    但是查看具体的授权到期日期，还是 2022年5月11号。

    再次登陆授权官网，发现已经将授权设置为了离线状态，无法修改授权了。

    后来我再次卸载，重新安装，尝试重新用这个账号进行登陆破解，结果以失败告终了。

    算了，只能是放弃了。

    参考文章:
    1.ArcGIS Pro 2.8.3完美破解版下载附安装教程 在线破解、在线验证后转移授权、离线破解，这三种方案，但是截图是ArcGIS Map的，百度网盘的安装包，只有一个17705的exe安装包，没有破解替代文件。
    2.Arcgis Pro中文版下载安装教程 （超详细附安装包） 这里的2.5的安装破解教程还是挺详细的，就是安装破解补丁这个地方不好搞。
    3.ArcGIS Pro 2.8.3 中文版本安装包下载 这里倒是有 ESRI_Licensing 文件夹的出处了，我下载的破解安装包里面也是有这个东西的，但是还是破解不了。
    4.地理信息系统软件 ESRI ArcGIS Pro 2.8.3 激活版 这里也有下载包，但是我没有尝试了，反正现在也用不到，算了。主要有：ArcGIS Pro 2.8.3 x64.zip、ArcGIS Pro 2.8 Data Interoperability.zip、ArcGIS Pro 2.8 Help.zip、ESRI ArcGIS Pro v2.5.zip 和 ESRI ArcGIS Pro 1.2.zip 这几个,用的是城通网盘存储的。

    isNaN 和 Number.isNaN 函数的区别？

    1.函数 isNaN 接收参数后，会尝试将这个参数转换为数值，任何不能被转换为数值的的值都会返回 true，因此非数字值传入也会返回 true ，会影响 NaN 的判断。

    2.函数 Number.isNaN 会首先判断传入参数是否为数字，如果是数字再继续判断是否为 NaN ，不会进行数据类型的转换，这种方法对于 NaN 的判断更为准确。

    为什么0.1+0.2 ! == 0.3，如何让其相等?

    一个直接的解决方法就是设置一个误差范围，通常称为“机器精度”。对JavaScript来说，这个值通常为2-52，在ES6中，提供了Number.EPSILON属性，而它的值就是2-52，只要判断0.1+0.2-0.3是否小于Number.EPSILON，如果小于，就可以判断为0.1+0.2 ===0.3。

    == 操作符的强制类型转换规则？

    1.首先会判断两者类型是否**相同，**相同的话就比较两者的大小。

    2.类型不相同的话，就会进行类型转换。

    3.会先判断是否在对比 null 和 undefined，是的话就会返回 true。

    4.判断两者类型是否为 string 和 number，是的话就会将字符串转换为 number。

    5.判断其中一方是否为 boolean，是的话就会把 boolean 转为 number 再进行判断。

    6.判断其中一方是否为 object 且另一方为 string、number 或者 symbol，是的话就会把 object 转为原始类型再进行判断。
