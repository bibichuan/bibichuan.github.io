    人生经验之借钱

    标签： 随笔 分类： 未分类 创建时间：2021-07-15 00:53:46 更新时间：2025-04-28 14:37:41

    最近啊，心疼的损失了8万块钱。这钱，不是大风刮来的，不是天上掉馅饼掉来的，而是两位老人辛辛苦苦用汗水一点一滴的摔出来的，这种坑人骗人的无耻之徒，但凡有点良心，也不会丧尽天良的把这笔钱吞下吧?我觉得可恨，却又没有一点办法，这钱拿不回来，我心里难过，却还是那么的无可奈何。我想着，就当自己救济灾民了，可是，这八万块，做些什么不好呢？最后连一句感谢，感恩的话都没有烙下，难道真的值吗？

    事情的背景是这样的，爸爸是搞长途运输的，开着那种长12米的板车，以前从山东拉货跑过杭州，跑过几次云南，现在年龄大了，已经55岁了，就单纯的跑跑南京了，从山东到南京，一来一回，加上中间休息，少则四天，多则五六天，有时候业主要的活比较急，就要连夜赶路，经常性的跑夜路，吃住都在车上，从没有下过宾馆，一趟下来，也就挣个两三千块钱。妈妈今年57岁了，觉得爸爸一个人跑长途，太累，平常没有说话的人，甚至连饭都吃不上，于是就陪着爸爸一起跑，虽然不会开车，但是起码可以陪着说说话，帮忙做个饭，有时候还可以帮着搬搬货物，盖盖篷布。十几米的篷布改在车上，其实很沉的，要两三百斤，特别是下雨的时候，一进水，更是沉的了不得，一个人根本就弄不动。我印象很深的是，有一次，爸爸说起来自己一个人盖篷布的时候，下雨天，没吃饭，货品又不能淋雨，淋了雨还要赔钱，自己一个人弄不动篷布，那时候都想哭了，你能想象吗？一个四五十的大人，就在雨里一遍干活一遍哭。每当想起这个事，每当说起这个事，爸爸的眼里总是擎满了泪花。母亲有一个哥哥,也就是我的二舅，二舅有两个男孩，也就是是我有两个表哥，大表哥大一点，2021年已经三十五六岁了，有两个孩子，大孩子已经开始上初中了，二儿子还小，具体多大我也不清楚。早年间，大表哥为了做生意，缺钱，想去银行贷款，但是因为银行不给贷，需要找到担保人，于是他找到了我爸，我爸给他担保借了银行20万。后来，大表哥做生意挣了点钱，但是因为自己好赌博，好玩乐，把钱都一点点的败光了，于是银行的钱也还不上了。银行不会吃亏的啊，于是就找我爸要钱，我不哪里能还的上呢？也都是挣多少，花多少的补贴家用。

    因为做了保人，第一次的时候，我爸把5000块钱放到了以他自己身份证办的银行卡里，后来我爸让我去取钱的时候，发现里面的钱没有了，后来找到银行柜台，说是因为做了保人，被担保人还不上了，就要保人去还。这是第一次吃亏。后来去找到大表哥，大表哥说，这事情已经摆平了，没事了，但是最后这5000块钱，也没有还回来。第二次的时候，我爸以为真的没事了，于是就陆陆续续的把钱打到了卡里，好几次的运费钱都打到了自己的卡里，这一次银行没有立马把钱转走，一直到年底，过了一年，我爸去取钱，发现里面的钱又没有了，为了银行，还是因为保人的事情，这个钱还是要保人还，这都是什么事啊。去找大表哥理论，大表哥就是一副死猪不怕开水烫的姿态，说自己没钱，还不了。也是因为心软，这件事情发生了，大表哥的大儿子生病了，要用钱，于是我妈就秉着先救人的原则，把一万多块钱又借给了大表哥，先让孩子把病看了。后来，孩子病好了，我妈去问要钱，大表嫂，说医疗报销还没下来，过两天再说。等医疗报销下来了，报了7000块钱，可是大表嫂只还了6000块钱，她说还有一笔报销，等那笔下来了，可以在还上。可是啊，后来我妈再打电话，连电话都已经不接了。

    本来我妈还抱有一丝丝希望，以为帮助了他们，自然他们也会帮助我们，可是最后一点希望也破灭了，连一万二的救命钱都已经不还了，难道还能指望他们还那八万的钱吗？银行都拿他没有办法，难道指望我们一介平头老百姓能拿无赖怎么办呢？八万块钱，这是何等的数目啊，为什么你们把钱拿去逍遥快活了，却还要让我们用命换来的钱，为你们偿还债务。你们的良心难道不痛的吗？你们难道没有一丝丝的良心吗？你们以及你们的孩子，难道从没有受过教育，就能心安理得的过下去吗？

    有时候，我就在想，要是能把这钱要回来，我该感谢他们全家人的努力，哪怕是少个一两万，我也愿意了，这就是借钱人最后的倔强了吗？我们只能怪自己当初好心做了保人吗？只能怪自己没有吃一堑长一智，还给他借钱吗？只能怪自己还是把钱放到了那张卡里了吗？为什么连亲戚都无法信任了呢？我拿这笔钱干什么不好呢？我买个房子不好吗？我买个车不好吗？让父母少干一天，少干一个月的活不好吗？越想，我都开始埋怨起父亲来了，为什么当初要给这样的人做保人，为什么第一次5000块钱被银行拿去了，没有把钱要回来，还是要把钱打到自己名下的银行卡里来，为什么还要相信恶人所谓的 “没有事情了”，转账之前，难道不应该先查一查吗？哪怕你在转五千，要是被银行没收了，以后的七万就不用再打了啊。为什么啊，为什么就是这么的单纯。可是难道相信别人，给年轻人一个希望，有错吗？现在后悔，也来不及了啊，如果他有钱，难道不会换银行吗？你这八万块，肯定也是要不回来，就算他洗心革面，认认真真的干活，一个月能拿个五六千，不就撑死了吗？还要照顾两个孩子，哪有什么闲钱，会给你呢？难道还指望他发财吗？那更加的不可能了，连贷款都还不上的人，怎么能发财？拿什么发财？那些努力的人都发不了财，难道一个吊儿郎当的人就能发得了财么？每每想到这些，我都觉得很绝望，很绝望，无奈，无助。我甚至有时候在怨恨父亲，为什么这么笨啊。我也甚至想到了，为什么当初大表哥在自杀的时候为什么不能死掉，如果当初他自杀成功的话，就不会有后来的贷款，担保，也不会有这两次的被银行收走八万五的贷款了。为什么老天爷还是让他活下来了，经历过生死，就能看淡事情，可是，他看淡的只是被人的性命和生活啊。

    另外一个发生在我自己身上的事，那就是我有一个小学同学，在小学的时候，我可以问心无愧的说，我很照顾他。他总是受人欺负，我作为班长也好，朋友也好，总是非常的照顾他，后来即便是初中毕业了，他不再上学了，我们还是保持着联系。后来一段时间，因为高中学业比较忙，也没怎么联系，等我上大学了，快毕业了，我已经开始实习了，那时候一个月只有3000块钱，好在我省吃检用，实习两个月下来，也还是攒下了5000块钱。有一天，不知道他从哪里知道我快毕业了，已经实习了，来找我借钱，说是借两千，我觉得关系也挺好，我也挺信任他的，于是就借给他了。后来过了几个月，他又来借，再借两千，于是我还是二话不说的借给他了，我也没有把他说的，月底还，或者年底还的话放在心上。就这样，我没有考上研究生，去实习的单位上班了，还是一个月四五千块钱。过去了两年多，借钱的他再也没有联系过我，我有一天，联系他说，能不能把钱还我，我要用钱。可是当时他就生气了，说自己没有钱，现在比较困难….后来我说，那你先还一点好了，于是就先还了我两千块钱，剩下的，说是下半年还清，于是我又等了半年。等我再次给他打电话的时候，他还是说没钱，在我不要脸的追问下，最后还是还了我1500，剩下的500块，愣是不愿意给了。我也是有恒心，有一次乘着从杭州回家的机会，我就开车去他家门口，要钱去了。当然，结果可想而知，不是赔礼道歉，不是热情款待，而是把我骂了一顿。最后，钱是要回来了，可是，从此两个人也就恩断义绝了。可笑的事，我从另一个同学那里听说，当他和这个同学聊到我的时候，竟是当着这个同学的面，不断的指责我，说是我的错。诚然，我做不会做人，不会交朋友，可是却还是因为借钱的事情，成为了一个被人指责的小人。

    为什么，受伤的，总是善良的人，我们从没有坑过别人的一分一厘，到头来，却还是被人骗的分文不剩。坏人总是逍遥自在，而好人却苦不堪言，这就是天理昭昭吗？拿起法律的武器吗？那我再给你说一个故事吧。我爸的一个朋友，姑且就叫大伯吧，大伯的妻子，也就是大娘，在几年前去世了，因为癌症。大娘去世的时候，还是念念不忘，拉着几个孩子的手说，“等我死了，你们要抬着我的尸体，到你舅舅家，问他们要钱，这个钱我们不能不要….”,我能想象的是，床前的孩子们听到这句话，该有多么的伤心 “妈，你就放心吧，这个钱我们会要回来的。” 这就是姐姐借给弟弟四十万块钱，到姐姐临死的时候，弟弟都没有还钱，甚至连姐姐去世都没有来看一眼的亲情。大娘生前，两家就已经开始打官司了，最后法院把弟弟家的房子判给了大娘，可是，弟弟一家人，死活赖着不搬，不走，有钱也不还，难道强行把房子卖掉吗？谁又敢买这样一个无赖住着的房子呢？这就是法律的正义，法律的光，对无赖，对恶人，无能为力，对坏人的无可奈何。

    世界上的恶人总是数不胜数，小到个人，大到国家，不也都是一个道理吗？最近比较热的 “ 林生斌” 事件，四年前老婆和三个孩子被保姆纵火烧死了，他自己用这件事博同情，获得了绿城一个亿的赔款，然后用老婆孩子死了的悲情男人，从淘宝店赚了两个亿，说好的老婆孩子用命换来的钱，不能花，可是最后还是花了，承诺的基金也没有成立，承诺的每一笔销售额的10%拿出来做公益，也没有做，甚至在老婆死之前就已经出轨了，还生了一个孩子，现在又和婚外恋情人结婚生了第二个孩子，已故老婆的家人，想要获取女儿留下的遗产，林生斌死活不给，用老婆的死赚来的钱，也一分都没有还给已故老婆的家人。从博同情，到赚钱，到谢幕，准备移民，这不就是一个赤裸裸的吃人的人吗？百年前的第一次鸦片战争，第二次鸦片战争，英法连接火烧圆明园，八国联军火烧北京城，中日甲午海战，日本第一次侵华战争，日本全面侵华战争……一桩桩，一件件，哪一个不是披着正义的外衣，行的是禽兽之事，到现在的日本，不也还是不承认自己犯下的罪行，仍然还是逍遥自在的过着富裕的生活吗？

    难道这八万块钱就无药可救了吗？难道只能哑巴吃黄连，再苦都要自己咽下去吗？难道就没有一个人能把钱要回来吗？苍天啊，为什么报应总是来的那么迟，那么慢呢? “正义，有时候会迟到，但永远不会缺席” 这个正义，我希望能在我父母的有生之年能实现，希望在我的有生之年能够实现。

    isNaN 和 Number.isNaN 函数的区别？

    1.函数 isNaN 接收参数后，会尝试将这个参数转换为数值，任何不能被转换为数值的的值都会返回 true，因此非数字值传入也会返回 true ，会影响 NaN 的判断。

    2.函数 Number.isNaN 会首先判断传入参数是否为数字，如果是数字再继续判断是否为 NaN ，不会进行数据类型的转换，这种方法对于 NaN 的判断更为准确。

    为什么0.1+0.2 ! == 0.3，如何让其相等?

    一个直接的解决方法就是设置一个误差范围，通常称为“机器精度”。对JavaScript来说，这个值通常为2-52，在ES6中，提供了Number.EPSILON属性，而它的值就是2-52，只要判断0.1+0.2-0.3是否小于Number.EPSILON，如果小于，就可以判断为0.1+0.2 ===0.3。

    == 操作符的强制类型转换规则？

    1.首先会判断两者类型是否**相同，**相同的话就比较两者的大小。

    2.类型不相同的话，就会进行类型转换。

    3.会先判断是否在对比 null 和 undefined，是的话就会返回 true。

    4.判断两者类型是否为 string 和 number，是的话就会将字符串转换为 number。

    5.判断其中一方是否为 boolean，是的话就会把 boolean 转为 number 再进行判断。

    6.判断其中一方是否为 object 且另一方为 string、number 或者 symbol，是的话就会把 object 转为原始类型再进行判断。
