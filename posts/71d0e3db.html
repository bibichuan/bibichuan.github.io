    软件研究之飞渡DTS平台

    标签： 无 分类： 未分类 创建时间：2022-12-29 12:03:06 更新时间：2025-04-28 14:37:41

    1.前言

    我最近的一个项目接触到了飞渡公司的产品，以前在各个开发者平台中，也有他们的人在宣传什么太极开发者平台，根据当时接触的那个开发人员来说，说整个的技术使用的是视频流的方式进行渲染，也就是说每一帧都是后端进行渲染和绘制，然后通过视频流的方式传输到前端，然后前端才去展示的，为此我还专门去了解了这个视屏流和像素流的关系。因为这个太极开发者平台，号称是开源免费的，其实是一个伪开源的，所有的渲染，脱离了他们的渲染平台都无法运行的，关键的渲染技术也没有足够的文档进行说明。在他们的 文档中心 中，我粗略的翻了翻，像服务发布，API接口这部分都没有相关的说明，唯一的说明，就是如何使用太极开发者平台这个软件，如何下载这个软件，如何启动这个页面，有哪些开发接口，如何进行二次开发这些都没有说明。

    经过一些群里的了解和看他们公司人员解决问题，这个平台主要使用的是WebRTC进行像素流的数据传输。

    参考文章:
    【1】.DTS数字孪生Paas平台 这里是飞渡的云渲染和开发手册，DTS平台软件包含Explorer、Cloud、Engine、Cluster和SDK五个产品。通过Engine把各种数据发布成3DT文件。Explorer把3DT组织成一个ACP工程。Cloud把组织好的ACP工程发布成云渲染服务。
    【2】.AirCityAPI AirCityAPI是一套基于JavaScript语言开发的二次开发接口，针对Web前端开发人员使用，用于在现代浏览器加载和展示由AirCityCloud引擎渲染的三维视频流，可以通过调用相关API接口与云渲染的三维场景进行便捷交互，目前AirCityAPI兼容当前主流前端框架，如Vue.js、React.js和Angular.js等。

    2.开发流程

    (1) 安装软件 Explorer、Cloud 软件,授权

    (2) 使用Explorer进行场景工程创建

    (3) 使用Cloud进行场景发布为服务

    (4) 使用AirCityApi调用场景，并进行渲染

    3.授权

    授权这个真是麻烦，销售也没有搞清楚，我还费了很大的劲。授权分为三类：免费试⽤账号(标准版）、FDN账号(专业版）、商业采购（硬锁，⽀持标准版/专业版）免费试⽤账号：通过官⽹使⽤⼿机号进⾏注册即可获得30天的标准版试⽤；FDN是⻜渡DTS提供的1年专业版授权服务，通过⻜渡商务可以申请专业版授权试⽤；硬锁是永久许可或订阅、专属密钥、⽆⽔印、专属客户成功经理⽀持。

    • 免费注册

      免费注册是标准版，试用一个月。

    • FDN
      官网注册之后，需要联系销售人员进行授权，才可以使用。

    • 硬锁
      就是类似一个 U 盘的东西，插到电脑上就可以在 CodeMeter 上识别，然后可以扩展授权。选择更新许可，生成一个 WibuCmRac 文件，发送给销售，他会给你一个授权文件，然后双击导入就可以了。

    isNaN 和 Number.isNaN 函数的区别？

    1.函数 isNaN 接收参数后，会尝试将这个参数转换为数值，任何不能被转换为数值的的值都会返回 true，因此非数字值传入也会返回 true ，会影响 NaN 的判断。

    2.函数 Number.isNaN 会首先判断传入参数是否为数字，如果是数字再继续判断是否为 NaN ，不会进行数据类型的转换，这种方法对于 NaN 的判断更为准确。

    为什么0.1+0.2 ! == 0.3，如何让其相等?

    一个直接的解决方法就是设置一个误差范围，通常称为“机器精度”。对JavaScript来说，这个值通常为2-52，在ES6中，提供了Number.EPSILON属性，而它的值就是2-52，只要判断0.1+0.2-0.3是否小于Number.EPSILON，如果小于，就可以判断为0.1+0.2 ===0.3。

    == 操作符的强制类型转换规则？

    1.首先会判断两者类型是否**相同，**相同的话就比较两者的大小。

    2.类型不相同的话，就会进行类型转换。

    3.会先判断是否在对比 null 和 undefined，是的话就会返回 true。

    4.判断两者类型是否为 string 和 number，是的话就会将字符串转换为 number。

    5.判断其中一方是否为 boolean，是的话就会把 boolean 转为 number 再进行判断。

    6.判断其中一方是否为 object 且另一方为 string、number 或者 symbol，是的话就会把 object 转为原始类型再进行判断。
