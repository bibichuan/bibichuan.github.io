    人生经验之京东购物

    标签： 杂文 分类： 杂文 创建时间：2020-11-09 13:19:20 更新时间：2025-04-28 14:37:41

    一年一度的双十一哟开始了，说一说我使用京东Plus购物的体验吧。马上就是双十一了，到底有什么好的呢？还不是因为穷，因为穷，所以才想着各种方法省钱，哪怕是省个一毛两毛，一块两块的，也觉得自己赚到了。

    1.全品类券的问题

    11月09日晚上一直到凌晨12点半还是没有休息，在干什么，在和京东交涉，为什么全品类券不能用呢？明明就显示的可以使用的。

    前几天11月4号，不是买了华为P40和华为navo 7 pro,我一口气下单了两件手机。当时以为很便宜了，反正也写着是价保，没想到今天(11月09号)更便宜，甚至navo也支持12期免息了，当时P40才支持6期免息。于是就想着把买到的这个换掉，换成可以十二期免息的。于是就下单了，可是没有了黑色的，只能买到了亮色白的了，我还想着，在上次购买P40的时候，不是顺手花了148块钱办了一个京东Plus会员吗，以前不知道，有京东全品类券可以使用，这次一并用上。
    根据说明和提示，使用了50的礼金，兑换了50的全品类券，然后点击立即使用。

    在领券的页面，明明P40这个单子显示可以使用全品类券，但是我去购物车付款的时候，还是没有减50，操作来，操作去，也没有给我减掉五十。最后无奈找了客服，开始呢，客服让我截图，然后找到了我的优惠券，然后就是订单，我还没下单呢。客服说是因为没货了，下不了单，所以用不了。我说我可以下单，可以付款的啊，于是我又是截图，又是付款，最后还是把单子没有使用优惠券付掉了，我就要证明我可以下单，但是不能用优惠券。我把我新下的单号，发给客服，这回客服换了口气，然后又去查了查，说是因为限品类东券和全品类东券不能叠加的。我靠，我当时就火了，我把他们管网上的使用说明，拍了一个又一个的照片，明明说的是可以叠加的，怎么到开卡之后说不能用了呢？

    反正我是没有找到那里说不能同时使用的。

    不是赤裸裸的诈骗，还是什么呢？我上网查了，很多人也都遇到了同样的问题，有的投诉成功了，有的没有。后来呢，说是为了提高体验，把我的券作废，然后给我补5000京豆，我本着不用白不要的原则，起码也是钱啊，就先让她办理了。
    可是我越想还是越生气，然后我想让她把我剩下的券都换掉，结果沟通无果，说对其他的顾客不公平，我抛开工作，也是能理解你的感受的。最后这个客服直接挂了我的聊天，连让我评价都没有。

    不一会，系统自动换了一个客服，给我解决。我再次说明是什么问题，为什么不能用，然后又是截图，简直就是让人抓狂，你们不能看看聊天记录的吗？为什么我还有再次的给你说一遍问题呢？我为了给她演示，我先是花了5块的礼金，换了以一个5块的券。她让我换一个商品试一试，我随便找了一个，还真的可以。就像第一个客服说的那样，没有限品类购物券的，这张全品类购物券都可以用。可是我已经没有那个单子了。

    最后我抱着侥幸的心理，希望她能像第一个客服一样，把我的券能换成京豆。于是我还特地把我剩下的45礼金统统都换成了全品类券，然后截图给她看，说这个也用不了，结果，就是因为买P40的有一个4488减200的活动不能用，再次下单的时候，没有了200的优惠券，于是就可以使用全品类券了。她让我换一个能用的，这不是扯淡的吗？我只想把剩下的40券换成京豆，客服不同意，说什么抱歉啊，没有权限啊，不能操作啊。因为已经12点半了，我实在是撑不住了，我不睡觉的？我懒得给她们争论了，最后还是想打个折也行吧,不给40，给个20也行啊，再不行，你倒是把这个40的券给我延长时间啊。全品类券，只要是换了就要在一天内用完，我怎么可能用的完呢，已经买了两个华为手机了。总不至于在买一个吧，40的全品类券要满1000元才能用，这得买什么呢？当然，天不遂人愿啊，客服不同意，既不给你退，也不给你延长时间。最后说，给我5元京豆了事。奋斗了一个半小时，扯了那么多的皮，最后只是给了55块钱的京豆最为提高体验的一次补偿。
    算了，算了，加上55元的京豆，加上30块的红包，加上电信的会员，就算花了几十块钱买了一个电信会员吧。

    此事告一段论。

    2.双十一价保的问题

    11月11日，我也是手贱，为了凑京东的满减，花了将近3个小时的时间去找，因为很多我需要的，比如抽纸啊，皮带啊，无线耳机啊，都不支持京贴满300减40的活动，还有图书啊之类的。怎么办，那就继续找呗，继续凑吧。最后好不容易凑到了，比如花了一百多买了个扩展坞，还买了砧板，不锈钢盆，还有杂七杂八的东西，最后终于满了300，减了40，加上我以前积累的京豆，用上之后，直接减掉了100块钱，最后付了150几块钱。然后又花了一个半小时，去天猫凑了三单，买了衣服啊，抽纸啊，还有小祝的东西之类的，总共其实也花了八九百块吧。

    双十一只剩最后半个小时了，我躺在了床上，我真是手贱，想起来还有京东的全民营业的红包没有用（突然想起来我在天猫下的单，也没有开奖啊，又错过一个亿），于是就点开京东，全民营业的开奖界面，开了个奖，也还好吧，开了16块钱，我以为只能双十一用，可是我已经买好东西了，看了看使用日期，可以到18号，还好吧，还可以在买些小东西的。
    然后，然后我又看了看刚刚到货，还没来的及开箱的两个华为手机的价格，一个是nava7 pro，一个是黑色的P40，一看不知道，看了才觉得自己亏了。P40还没有怎么降价，虽然成交价为4278，比我的便宜了10块钱，但是nava7就不一样了，竟然除了有12期免息，还加上了300的优惠券，还有就是又多加了一个50块的优惠券。

    这是我当时的订单。

    这个我就受不了了，我好不容易用红包换来的省钱，就这么白白的浪费了吗？不是说好的价保的吗？怎么又出现了这个情况？于是又开始了我的维权之旅了。

    时间已经来不及了，12点50分，我先结了个图，然后去找客服理论。刚开始，客服还是说，没货了，下不了单。然后就是让我先领券，下单，然后把我的券给我作废掉，返还给我京东，又是这个套路。客服让我先领券，然后截图给她。我操作来，操作去，已经到了12点半了，已经过了12点了，没有地方领券了啊，而且价格已经回到了3699。这让我我怎么领券呢，怎么下单呢，而且也没货啊。感觉又被耍了一波。我说我领的券已经没了，她又说，没有办法了。我于是就找了一个其他的券，截图给她，然后他说看不见，让我下单，不付款。不付款就下不了单啊，她说”那就是不支持使用的呢，没有办法价保呢“。最后我还是下单了，买了一个其他的手机，然后截图给她，让她把这个券给我免掉。她说不行，“这个手机和之前那个手机不是同一个呢，您是价保之前那手机，需要用之前那个手机下单呢”不是一个手机，不行。不是一个手机，你不早说？我领的券是真的，截的图也是真的，他们又不是不知道，就是不愿意给补插价。兜兜转转，最后又弄到了一点钟。我气氛的睡觉了。

    早上起来的时候，我还是觉得这是什么鸡巴购物体验呢？我不服气啊。我不甘心啊，于是又去找了客服，我非得说个明白不可。而且我也咨询了伟大的群友，群友说，去投诉，去投诉，去投诉。我连过了价保的都可以给你退回来。而且他已经成功了。

    本着不能吃亏的原则，我还是厚着脸皮去和客服扯皮了。客服还给我截图，说“您选购的此款热销商品已无货，暂不支持价保哦，请您理解。”理解，我怎么能理解。无货，那等有货的时候，价保不就过期了吗？这是什么逻辑？这是你们的问题啊。因为你们的宣传问题，而且还明写了双十一的时候同价，现在出现了降价，让我怎么理解你？你能不能理解理解我啊。最后，客服说：关于您反馈的问题这边去帮您核实24小时内短信告知您结果。我在等24小时吧，希望有个好的结果。

    说好的24小时给我回复的，虽然没到24小时，我反应问题是在早上九点中，现在已经到了下午六点中了，还是没有消息，也没有短信，我猜，一定要等到我明天再次去问的时候，才能给我消息吗？这处理速度，也是没有谁了。

    晚上17:40，京东给我发了条消息：【京东】尊敬的客户，我是京东客服，（关于您反馈的订单137965843111价保问题，这边已经帮您核实，确实无法申请差价，需要在降价的当时申请，价格保护作为京东的特色服务，也是充分在我司可承受范围内保障客户的利益，对于规则范围内的降价，我们会积极为您申请。但也请您理解市场价格波动）。祝您购物愉快！

    这是什么话，必须降价的当时申请，我申请啦，你不同意啊。又让我等24小时。申请价格保护，要在当时申请？你们降价的时候也没给我说啊，现在我是看见了，你们把价格调回去了。降价的时候怎么不给我说一声？问题我当时就提出来了
    你们不同意啊。明明双十一有活动的，商品上会价保的，无论是华为的价格还是你们京东的优惠，不都是应该在价保范围吗？我要是看不见，你们就不赔了吗，就可以省掉这笔钱？核实了，还是说没有当时申请，我当时申请了，谁知道你们速度更快啊。是的，为什么双十一的价格会更低，不给我退？京东客服：妹子要以现在的为准的呢。这能让我信服吗？你们不就是打着双十一的活动吗？又让我等，在等24小时，简直了。

    你说，这真的是我的问题吗？明明是价保，就要一直是这个价格啊，我是在降价之前买的啊。无论客户看不看的见，知不知道，不都应该是这个价格吗？这次是价格保护的含义啊，这才有意义啊。最起码，降价了，你得通知一声啊，也好让我主动的发起申请啊。如果顾客信任你，我就知道这时候是最低了，一般不会在去纠结了，你突然就降价了，没给顾客说，顾客发现不了，就以为是最低价了，这样你就心安了吗？现在是我发现了，而且有证据了，你们还是不承认，还是拿现在的标准来说，这怎么能让我信服啊。我发现的当时就申请了，只不过是因为时间短，没来的及罢了。

    等待，漫长的等待，最终的结果是什么呢？我会如愿拿到我要的50元的京豆吗？作为弱势的我来说，估计够呛啦。

    11月13日，京东来短信说：【京东】尊敬的客户，我是京东客服，（您好，您咨询的差价的问题核实是没有降价的呢，非常抱歉的，给您不便了）。祝您购物愉快！这不是明摆着坑我吗。再闹下去，可能真的没意思了，最多就是退钱，退款呗，哪里还用得着这样呢？浪费时间，浪费精力给京东瞎扯淡。

    3.投诉

    无奈啊，京东客服还是不予理会，我还是觉得自己受到了莫大的委屈，我只能孤注一掷的去投诉了，虽然不一定能成功，但是我还是要努力一下的。有人从黑猫投诉平台上进行投诉，我因为微博账号的问题，没法登录微博，于是就找了国家互联网信息投诉平台上进行投诉,填写了相关投诉信息。

    最后等待结果好了。

    作为京东方面，他可以随意调整价格，今天降了价，明天就恢复，等顾客发现的时候，说：谁叫你不早点来呢，我们已经恢复了原价了，这个你没话说了吧。如果你降个两块三块的，我也就不说什么了，但是你这是50块啊。如果顾客没有发现降价了，难道就不用价保了吗？你辜负了我对你的信任，为什么就不能痛痛快快的给我也降价呢？真是服了。这次的体验，无论成不成，都让我对京东失望极了。

    除了国家的投诉平台，我也申请了一个账号，去黑猫投诉平台进行了。我11月4日买的手机，宣传页，图片以及详情等，都说会有价保。我就安心的等着收货了。刚开始我也就没有在意，没有发现，11月11号的11点半我去看我买的华为navo7,发现降价了，申请价保，京东不同意，说我没有当时就提出来，已经过期了。我截图了，也不行。

    京东的宣传存在虚假宣传，既然价保，为什么要等着我去发现呢？我不发现降价了，就可以不履行价保了吗？没发现就可以不用价保了，难道我还要24小时盯着吗？那价格保护给我的安全感在哪里呢？和你宣传的可以放心购买不是一毛钱的关系也没有吗？

    还是希望能有个好的结果，能引起更多人的重视，我相信不止我一个人是这个骗局的受害者。在知乎上直接搜，京东价保，就会发现很多人描述的情形给我的是一摸一样，都是出现了这样的情况。

    isNaN 和 Number.isNaN 函数的区别？

    1.函数 isNaN 接收参数后，会尝试将这个参数转换为数值，任何不能被转换为数值的的值都会返回 true，因此非数字值传入也会返回 true ，会影响 NaN 的判断。

    2.函数 Number.isNaN 会首先判断传入参数是否为数字，如果是数字再继续判断是否为 NaN ，不会进行数据类型的转换，这种方法对于 NaN 的判断更为准确。

    为什么0.1+0.2 ! == 0.3，如何让其相等?

    == 操作符的强制类型转换规则？

    1.首先会判断两者类型是否**相同，**相同的话就比较两者的大小。

    2.类型不相同的话，就会进行类型转换。

    3.会先判断是否在对比 null 和 undefined，是的话就会返回 true。

    4.判断两者类型是否为 string 和 number，是的话就会将字符串转换为 number。

    5.判断其中一方是否为 boolean，是的话就会把 boolean 转为 number 再进行判断。

    6.判断其中一方是否为 object 且另一方为 string、number 或者 symbol，是的话就会把 object 转为原始类型再进行判断。
