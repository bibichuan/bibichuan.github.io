    人生经验之某公司

    标签： 无 分类： 未分类 创建时间：2023-04-19 07:58:31 更新时间：2025-04-28 14:37:41

    前言

    这篇文章记录了我在一家小公司中经历的种种问题，我想在其他的公司中也会有这样那样的问题，但是最终选择是不是留在这个公司，只有两个原因：
    1.钱，有没有到位。
    2.公司是不是有上升空间。

    很多人都会告诉你，在一家公司，不能总是看到这家公司的不好，要看到公司的好，要努力的适应公司的文化，要怎么怎么样才能获得职场的晋升，否则就会被淘汰，等等，这样一系列的观点和建议。而我，作为一个没有那么多上进心的普通的员工，看到的感受到的，就是公司有没有真的把我当一个员工看待，有没有给够我足额的薪水。毕竟啊，现在这个社会，房价这么的离谱，我还是要生存的啊。

    1.大小周事件

    2021年1月和2月，原先的政策(该政策已经决定了一个多月了)为2021年1月的第一个周末为双休，这样1月9号、1月10号修两天，1月23日、1月24日休两天，2月6号也可以修一天，在2021年01月07日突然出了新的措施，调整休息的策略，老板说有长放假或小长假后一般是安排单休，1月16、17号，1月30、31号休两天，但是这样的化，2月6号就要上班了，2月份一直要上到9号，才能放假，这不是明摆的多上了一天的班了吗?然后美其名曰，不是为了多上一天班，才调整的，是为了休假的合理性，你TM不是为了多上一天班，你到是把这天放掉啊。经过沟通之后，他说你请假，给你造成的不便，我自己把这个前替你还上，当然，最后也没有兑现。

    2.无偿加班

    总会让你无偿的加班，周六周日有时候也会给你打电话，当然，你可以努力，但是不会有丝毫的效果，加的班，也不会让你调休，也不会给你一点加班费。

    我还记得有两个实习生，因为下午下班后出去吃了个饭，然后就被克扣了晚上的餐补。

    还有一次，我记得有一个产品经理级别的人，因为晚上下了班去吃了个饭，然后回来被他看见了，于是他说，我不是推荐给你了外卖吗，你可以点外卖啊。我刚进公司的时候，那时候是干到晚上六点半就可以有30块钱的餐补，后来，因为我经常都只到六点半就走了，于是，他有一天在群里说，以后晚上干到七点才有餐补，这个我也忍了。2021年02月04日，小年，我在5点40的时候，准备下班了，他竟然非常大声的对我说，你给我回来，当着全公司的面，诚然，我还有工作没有完成，但是我一平常都是要到七点才走的，偶尔早下班，怎么了呢？

    今天我五点半下班了，又出来给我找活干了，我明天就要出差，明天就要给客户展示，明天怎么怎么怎么样，可是呢，等明天到来的时候，还是没有预想的什么项目啊，领导啊，开会啊什么的。

    3.软件团队不全

    所有的事情都是我来做，包括前台、后台、测试、数据库维护，这都是一个人的活。很难想象在这样一个自己就是天的团队里，个人能有什么样的成长呢？有一天，他出去谈客户，拿出了不成熟的软件，最后呢软件功能不全，出现了一个按钮无法点击的情况。于是就立马打电话把我骂了一顿，大致意思就是，你是怎么做事情的？为什么这个功能没有实现，为什么出现了错误，这很丢人好吧，我们把软件给别人用，这个不能点，那个有错误，很丢人的好吧，我拿给别人看，这是在打我自己的脸，你知道吗？你什么时候发现这个问题的？为什么不提前通知？不要什么都等着我邱总来发现好吧，责任心有没有的？如果我不发现，是不是就当什么事情都没有发生呢？诚然，我个人的能力还是有很多的缺陷，不能编写健壮的代码，不能对软件进行充分的测试，没有及时报告问题的原因。话又说回来了，这个功能按钮不能用，你不会不点击它吗，用手输入不照样也是可以的吗？简直是蠢到了极点。

    总是说不是不愿意招产品经理，是招不到，不是不愿意招技术大牛，是因为没有，我们的要求太高了。这些话，我都听烦了。

    4.让人写检讨

    任何的问题都得有人抗，但我不愿意这么抗。老板喜欢让别人写认错说明，就是从什么到什么的，说你这个不行，那个不行，这里不行那里不行。我还记得在2020年春节的时候，一个年轻的人事，因为一件什么事情，他就让这个人事写了一份长长的认错说明，第一遍不合格，还让他写了一遍，可是最后呢，还是在过春节之前把他辞退了，当然了，春节的钱也没有发了。 在资本家的眼里，这是正常的操作，为了节约开支吗，可是对于一个普通的员工呢？幸亏这个员工还是比较年轻的，还可以从头再来，如果一个老员工呢？该如何才能生存呢？

    2020年春节的时候，有一个叫王xx的小伙子，是一个人事。不知道因为什么东西没有做到他心里去，最后就是要他写了3000字的认错说明，可是最后还是在过年之前就把人家辞退了，连一分钱的补贴都没有给，但是还是给我们说，是人家做的不好。

    5.离职时总会和人吵架

    最近的一次一个实习的小姑娘的事件，小姑娘刚毕业，来到了实习了一个多月，因为家庭的原因，于是提出离职，去他的办公室，回来的时候就被骂哭了，说人家不尊重他，凭良心说，其实小姑娘从入职以来，根本就没有和他说过几句话，谈不上什么尊重不尊重的。难道什么是尊重？离职都不行了吗？最后离职的时候，非得让人家说明离职的理由。等小姑娘走的第二天，他就神秘的过来给我说，她离职的原因是因为怀孕了，要去医院打胎,我真是醉了。

    没有一个人走的是安详的。我时常都这么对离职的人这么说，你走的很安详。最近的一次秦离职，在送他走的时候，本以为他走的很安详，没有什么问题。可是周五走，周一就出问题了。因为老板不给批离职证明，而秦又需要这个证明入职一家新的公司，结果，就去催老板，最后老板知道了是因为有了好工作才离职的，而不是因为家庭原因而离职的，所有就非常的生气，非常的生气，于是等秦来拿证明的时候，就是一顿的猛吵，最后还让人事部门以及负责人章进行了检讨说明，一直加班到晚上九点半写说明。无论处于什么原因，最好都是好聚好散。可是在这个公司不一样，无论怎样，都会吵架。还记得上一次那个离职的刚毕业的小姑娘，去离职的时候，竟被他给骂哭了，说人家不尊重他，说小姑娘脑子有坑。说句良心话，小姑娘从进公司到离职，一直都没有什么出格的事情，也不存在什么尊重不尊重的问题。可是，就是最后还是要骂人家，我就是搞不明白了。

    周走的时候，因为是做硬件开发的，还要人家签一个什么协议，说是拿着技术去赔50万，从公司挖人赔100万。除了给人使绊子，还会威胁人，最近走的这个算法的小姑娘，就被说，整个杭州都是浙大系的，你去那个公司我都可以给你搅黄了，难道他们不做备调的吗？

    还挺跟了他八年的老员工说，招人来的时候，都会说人家多么牛逼多么牛逼，可是没多久就被人家告上了法庭，他给我数了六个，都是这样的情况，来的时候好好的，走的时候，不是吵架就是对薄公堂。

    6.迷之自信

    老板总是有一种莫名其妙的自信，觉得自己什么都能干。任何问题在他手里，三篇论文就能搞定。对外吹嘘，自己是首席科学家，贝尔实验室出身，可是实际上，就是一个浙大的本科生。说自己参加过鸿蒙系统的开发，鸿蒙系统是他开发的。还说自己写过操作系统，当时三星的老总找他，让他担任浙江地区的一把手，他没有同意。最近的一次就是说，关于地图避让算法的研究，我本意就是不愿意实现，在我的印象中，ArcGIS都没有实现的功能，他一个电话，非得要我实现这个功能。其实我非常的讨厌这样的人的，总觉得自己可以掌控一切，完成一切，甚至有些不能听取别人的意见了。总是时常的说，没有我做不了的事情，你告诉我坐不了之前，得拿出一个方案来。理论上是没有错，但是我非常讨厌这种感觉。任何做不了的事情，我都要给你说，然后就是要写一篇文档给你，非常的不喜欢。

    7.工作生活分不开

    总是会在你吃饭的时候，下班的时候，给你打电话，给你安排任务，没做完你就加班。周末的时候也会给你打电话，让你干活。你会有一种在休息的时候，还会被老板支配的恐惧。

    老板自己总是早睡，然后四五点重就醒了，于是就可以给你发消息，发短信，甚至有时候会在你下班的时候就给你打电话，所以你不得不在吃饭的时候，就要接他的电话，在下了班回家的路上接他的电话，回复他的信息。是啊，我作为一个人，就要把所有的精力和时间都花在你这个公司上面吗？

    8.喜欢背后说人坏话

    我刚来的时候，有一个合伙人，叫沈总，刚开始和老板两个人好的那叫一个人，总是互相吹捧，互相拥抱，可是后来，不知道因为什么原因，两人分开了，于是就时常在我们面前提，说沈总如何如何的不好，然后坑了他很多钱，挤兑公司里的其他人。还有他还招过一个叫李总的人，来的时候，也是给我们吹牛逼，这个人和他交情多深多深，能力多棒多棒，没过一个月，找各种理由，把人家挤走了，好像用的是日报不合格，挤走他的那段时间，他疯狂的抓日报，日报的字数要达到多少，内容要怎么怎么样。等这个人走了之后，每次开会，经常提起这个人能力不行，不能为公司带来人脉，不敬业之类的。还有就是胡建华，也是，说他不诚信之类的…反正啊，好多人来的时候，他都不遗余力的吹嘘这个人的能力多好多好，可是一旦这个人离开了，那么所有的错，所有的不人道，都是这个人的，会把他贬低的一无是处。我能想象到，我离开的时候，也不会落下什么好话。那也无所谓了。

    9.产品经理思维混乱

    他常说的一句话就是，你们要有产品经理的思维，何为产品经理的思维呢？今天他想了一个功能，说是基础功能，明天他又想了一个功能，说是基础功能，所有的都是基础功能，那还有什么不是基础功能呢？关于地图标签的自动避让功能，给我来了一句，这是基础功能，一定要做，等我手动完成了拖动标签进行修改的时候，他说这是反人类的设计。我真是不明白了，到底什么才是当前最重要的呢？

    10.年会奖品不兑现

    我找不到更加文明的词语来形容这个事情了。2021年01月23日，公司组织了团建，组织了年会，在酒桌上，我们极尽吹捧之能事，阿谀奉承老板。老板也很高兴啊，然后就是抽奖，说好的一二三等奖，有什么，有什么，抽完了，我还得到了一个五星级酒店的报销，最后告诉我，要明年678月份才能用，那还叫年会的奖项吗？年终的福利还有什么意义呢？还有其他的，比如买个榨汁机，买个耳机什么的，都说好的，可是万万没想到，原来喝酒可以醉人，不喝酒的人，也会自醉，等年会结束了，新的一天到了，等我们把需要的东西报上去之后，最后就如石沉大海，毫无音信了，除了拖，还是拖，现在已经是02月02号了，还有八天就过年了，还能拿到那2021年的礼物吗？对公司有什么期待，我可能真是瞎了。

    11.人员流动大

    人员流动，本来是一个公司常有的事，即便是我以前的事业编制，每年也会有人进来，有人离开。可是这个公司的人员流动频繁到什么程度呢？在我入职一年半的时间里，我已经不知道送走多少人了，最后，软件开发的，就只剩下我一个了。我回想一下。看看还能记住多少人名，还有忘记名字的，还有来了一天的，一个小时的，两个小时的，都有，有些人走了，连名字都没有留下。这些人，有的走的很安详，有的人走的很不安详，大部分都不是安详的。还有一个人，我虽然从未遇见过，但是他加了我QQ,说自己是前年在谱恒工作过的，被坑了很多钱，也说了老板很多的不正当行为，并力劝我早日离去。我要为这个不知名的朋友，送上祝福，他说的每一句话，都是实实在在老板做过的让人不齿的事情，甚至还有相应的截图，我忘记保留了，很可惜。

    2021年4月08日，最近一次的人员离职，是两个刚毕业的研究生，其中一个被扣发了一个月的工资，到现在也没有结清，经过多次沟通和协商之后，还是不发，只能申请劳动仲裁。第二个人，在提交离职申请之后，迟迟不予办里离职手续，硬生生的拖了一个月，到现在还是不予办里手续，美其名曰是因为没有交接完工作，但是实际上，是因为没有人接收其工作。

    12.没有公司福利

    (1) 节假日福利
    公司福利？可以说没有，端午节，春节，什么节都没有，中秋节的时候，倒是有购物卡发。2021年的春节，老板很小气，最近不是买了东西寄回家吗？还神秘的说是大礼包，很好的东西，给父母的，不让我们经手。经过快递回家之后，我妈都嫌弃，说，顶多一百块的钱的东西，还值当的送回家，还不够邮费的。

    (2) 请假
    平常的节假日也没有什么福利，节前和节后请假，还要扣双倍的工资，这一点就有点受不了了，凭什么要扣双倍工资？我干多少天，给多少天的工资，为什么要多扣我钱呢？

    (3) 年假只有三天
    入职满一年之后，到第二年的入职日，有三天的年假，只有三天，你不请，也不会给你加钱。所以啊，不请白不请。

    13.年终总结

    在开年的会议上，他重提了写年终总结的事情，说有些人年终总结写的长篇大论，写这些没有用，难道我不知道你们这一年来干了什么吗？你们要写你们以后能为公司做什么，明里暗里的讽刺我。看不惯我的年终总结，你明说啊，既然年终没有用，那就不要让写，我写了加工资30%,也没有加啊？这个你怎么不提呢？

    14.个人无法适应公司的模式

    其实我非常的受不了这种形式，比如昨天提出了一个功能，就是要在c软件内嵌的一个浏览器(应该是谷歌浏览器)中进行登录跳转，于是我了/user/clientlogin?client_id=s&client_secret=d 这种方式开发了，可是等第二天的时候，又出现了新的需求，说这样不行，要把client_id=s&client_secret=d进行base64编码，于是我换成了/user/clientlogin/{base64}格式，于是我写完了发给他，在多人电脑上测试都是没有问题，但是他又不行了，因为他用到了cef浏览器(一个内嵌在桌面应用程序中的浏览器)，死活加载不出来，一直显示空白，于是就说我写的程序有问题，把我披头盖脸的就是一顿臭骂，你能不能测试好了再发给我，不要浪费我的时间好不好？至少你要在其他人的电脑上测试好再给我的啊，这个功能本来昨天就要测试好的。我心理真是有一万头草泥马在奔腾啊。功能我昨天就发你了，你自己不测试，测试出现问题，回过头来还要怪我，临时改变了需求，又要急着上线，完不成还一个劲的电话催，谁爱伺候谁伺候，我不干了。

    15.整天瞎几把忙

    老板总是标榜自己很努力，晚上只睡四个小时，其余的时间都是在干活，这很令人佩服。但是我不认可这种努力，作为一个老板来说，还在自己亲自编写软件，但是这个软件并不是什么核心的科技，比如桌面软件中嵌入一个Cef浏览器。
    做事情没有轻重缓急，作为一个产品经理来说，作为一个需要进行开发的人来说。总是会出现这种情况，就是主要功能没有，最后都会纠结于一些不足轻重的小问题上来。
    (1) 第一就是我上面提到的事件，使用带问好的方式和不带问号的方式，其实我已经上交了相关的功能设计说明，也已经找他确认过了，最后还是会出现功能反复的问题，这就让开发人员很头疼了，明明已经按设计书进行开发了，结果还是会临时变动，就是因为他完成不了接下去的功能。

    (2) 还有很多这种事情，比如浏览器缩放问题，浏览器不是有一个可以将当前页面缩放多少倍的功能吗，我开发了很多的年了，都没有用过这个功能，他非得说这是一个常用功能，让我把功能也就是地图上的label标签的大小设置成随着浏览器缩放倍数自动更改的功能,虽然花了很大的力气搞完了这个东西，但是最后呢，还是没有用上。

    16.言而无信

    给我的感觉就是这样的，言而无信，工资总是一拖再拖，说好的是十五号发工资，但是几乎从来都没有准时过。年会上(如果那也算是年会的话)说好的会发奖品的，最后一拖再拖，2020年已经过去了三个月了，到现在还是没有影，最后告诉我们，说你们自己买，自己买，然后回来报销，最后能不能报销，都已经是一个迷了。餐补已经拖欠了三个月了，去年12月份的就没有发，一月份，二月份，都没有发，这让我们还怎么加班呢？

    不知不觉，已经在这里干了三年了，按说应该提前把合同准备好吧，可是老板不会，老板总是喜欢拖着，拖着，就算你去问了，还是给你拖着，我就是不明白了，哪怕是按照三年前的合同再重新签一份，不很好了吗？非得要改来改去的。老板说：“劳动合同是存续的，在特殊期间，不用催。” 合着这个意思是签不签劳动合同都可以？后来我查了查，看了看网上的说服，“劳动合同到期后，用人单位和劳动者没有续签，劳动者还在继续工作，按照《劳动合同法》规定要求，超过一个月不满一年未与劳动者订立书面劳动合同，劳动者可以要求支付从第二月起支付每月二倍工资。自用工之日起满一年不与劳动者订立书面劳动合同视作签订无固定期劳动合同，用人单位单方面解除劳动合同，若劳动者无过错情况下，用人单位属于违规解除劳动合同，需要按照《劳动合同法》第四十七条规定的经济补偿标准的二倍向劳动者支付赔偿金。” 和老板打交道，没有一个律师朋友，可真是举步维艰啊。

    isNaN 和 Number.isNaN 函数的区别？

    1.函数 isNaN 接收参数后，会尝试将这个参数转换为数值，任何不能被转换为数值的的值都会返回 true，因此非数字值传入也会返回 true ，会影响 NaN 的判断。

    2.函数 Number.isNaN 会首先判断传入参数是否为数字，如果是数字再继续判断是否为 NaN ，不会进行数据类型的转换，这种方法对于 NaN 的判断更为准确。

    为什么0.1+0.2 ! == 0.3，如何让其相等?

    一个直接的解决方法就是设置一个误差范围，通常称为“机器精度”。对JavaScript来说，这个值通常为2-52，在ES6中，提供了Number.EPSILON属性，而它的值就是2-52，只要判断0.1+0.2-0.3是否小于Number.EPSILON，如果小于，就可以判断为0.1+0.2 ===0.3。

    == 操作符的强制类型转换规则？

    1.首先会判断两者类型是否**相同，**相同的话就比较两者的大小。

    2.类型不相同的话，就会进行类型转换。

    3.会先判断是否在对比 null 和 undefined，是的话就会返回 true。

    4.判断两者类型是否为 string 和 number，是的话就会将字符串转换为 number。

    5.判断其中一方是否为 boolean，是的话就会把 boolean 转为 number 再进行判断。

    6.判断其中一方是否为 object 且另一方为 string、number 或者 symbol，是的话就会把 object 转为原始类型再进行判断。
