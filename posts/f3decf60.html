    人生经验之不孝子

    标签： 无 分类： 未分类 创建时间：2023-08-08 14:16:28 更新时间：2025-04-28 14:37:41

    前言

    人这一辈子，短短几十年，要讲良心，丧尽天良无恶不作无异于禽兽，活着只能算算是垃圾，死了也会不得到安宁。五千年的文化，教育我们做人要讲孝悌之道，何时何地这个道理总是有用的。谁都会老，人在做天在看，你如何的对待父母，将来你的孩子也会依样画葫芦，有样学样，成为下一个不孝子，你父母经历的磨难，也会同样上演到你的身上，这是因果，也是报应。我想写一篇文章，痛斥那不仁不义，不孝不忠的人，可是我无法用语言去形容这种人的所做所为，这令人发指，令人恶心的行为，即便是说一千遍，哪怕经历了千年，依然应该让人不齿。

    什么叫作罄竹难书，就是一个人做的坏事太多了，把所有的竹子上写满了，都无法写完。奎东和她老婆，就是这样的人。像奎东和他恶毒的老婆，伤害的是爱他，疼她，关心他，帮助他养了他一辈子的最亲的父母，顺带着，还连累了他的兄弟姐妹。这样的人，还有活在世界上的理由吗？我记得《济公》有一集，讲述的就是一个地主家的纨绔儿子，不仁不孝，老爹病卧在床，他非但不照顾，还为了家里的财宝，要亲手杀了自己的亲爹。相反的就是那个无钱无势的穷人家的孩子，即便不是自己的亲娘，知道是这个女人含辛茹苦把自己养大，也还是尽心尽力的照顾，两相对比，什么是孝，什么是不孝，不言自明，高下立判。隔壁的哑巴姑娘，心地善良，还知道时常的来看望照顾生病的老人，自己的亲儿子，连一个外人都如，要他有何用呢？最后这个不孝子被济公变成了狗，而孝子和哑巴姑娘也喜结连理，有了一个和满的结局。是啊，一直一来的奎东，就是那条狗都不如的畜生啊，连毫无血缘关系的外人都能知道时常来看望老人，他连自己的亲娘都不认，这是怎样的铁石心肠呢。但行善事，终会得福报，我始终无法明白的就是，如此阴险的小人，何以活到了五十多岁，何以有工作，何以养孩子。

    我奶奶一生与人为善，不曾和什么人有过争吵，甚至极少会听见她骂人的时候，勤勤恳恳兢兢业业，八十多了，却还要被不孝的儿子折磨，老天爷真是不厚待善良的人。我爷爷这辈子，20岁之前靠母亲，70岁之前靠老婆，到了80岁，还要靠儿女，享了一辈子的福，却也没有为身边的亲人带来福祉。这个世界本就不是公平的，好人不长命，坏人横乡里，上一辈有上一辈的恩怨，我无法代替他们去解决问题，我唯一能做的就是将奎东及其媳妇的所作所为用笔记录下来，作为警醒自己，如果有幸有孩子的话，也作为警醒后辈的例子吧。

    1.泼辣凶悍

    2005年的时候，那时候钱还是很值钱的时候，爷爷没有工作，已经六十多了，好不容易到了村西头的煤矿着了一个装煤的工作，辛辛苦苦干了一个月，等到月底去领工资的时候，结果被告知说是被二儿媳妇领走了。于是爷爷就去问儿媳妇要钱，儿媳妇说：“你的钱就是我的钱，我没钱了，拿着用了，要钱没有。” 爷爷打不过，也说不过，就在门口站着不走。奎东的媳妇见状很生气，就直接拿出领到的五百块钱，然后当着爷爷的面，死的粉碎，一把扔在了爷爷的脸上，“你要，你拿去好了，赶紧滚。” 懦弱的爷爷就默默的捡起碎了一地的钱，拿了一个袋子装起来，带回了家。一个月30天起早贪黑的努力，就随着钱的粉碎，就这么没了，碎掉的还有那个千疮百孔的心。

    2.打爹骂娘

    一个恶毒的人，能恶毒到什么程度呢？自己是一个贱人，就觉得所有人都不是好人。那时候奶奶还能干活的时候，在自家后院里种了点茄子，每天悉心的照料，不断的呵护。等快要熟了的时候，想着能一家分点。有一天奎东的媳妇下班路过，看到了菜园的菜不错，于是就进去看了看，然后顺手摘了一个茄子。奶奶出门打水浇地，刚好看到了这一幕，于是就说：“你先不要摘，现在还不熟，等过两天在摘。” “啊，我就要摘了怎么了？是你众的？你种的我摘点怎么了？” “不是不让你摘，过两天长的好点，你再来不就行了吗？你现在弄了，把其他的都弄坏了，没法长新的了。” 奎东的媳妇还是不听，还在摘，把那些刚刚开花的，刚刚结果的也都摘了，能吃的就留着，不能吃的就直接扔掉。奶奶于心不忍了，于是就伸手去阻止她，见状，她更来气了，一下子掰开了奶奶的手，还不忘把奶奶用力的推了一把，奶奶直接就坐在了地上。奶奶艰难的撑着地面，起来，她又一次伸手，把奶奶推倒了，“我吃你点东西怎么了？你这么小气。” 然后甩甩袖子，带着自己的战利品走了，留下的就是奶奶无助的身影，还有隐藏的泪水，看着自己辛辛苦苦的劳动成果，还没有看到结果，就夭折在了襁褓之中。

    3.胡说八道

    我总结了下，她之所以是谎话连篇，是因为她只有不断的重复不曾发生的事情，不断的把自己的罪恶说成人家的罪恶，这样才能让她睡的安心，睡的踏实，因为说的多了，似乎就不是她做的了，而不是别人做的一样了。口口声声的说父母没有给她看孩子，自己做月子的时候，想吃点面条，都没有人给做，还是儿子回家来，自己才起来做的,要不就饿死了。我还能清晰的记得，奶奶拖着沉重的身子，不断的抗争：“没给你看孩子吗？怎么没给你种地？怎么没给你做饭？” 奎东的媳妇，就是仰着头，一个劲的否认着，否认着：“没有，就是没有，就是没有。” 仿佛她越大声，这样的事情就会变成真的一样，她说的话就成为了现实一样。

    作为一个正常人，始终无法理解神经病的思维。奎东的老婆总是不断的给别人说，自己多么多么的孝，别人多么多么的不孝。总是在说别人占了她家多少的便宜，吃了她的，喝了她的，用了她的，自己什么都不计较，不明事理的人听着这么义正严辞的话还有超级认真的表情，都会忍不住的同情她，宽慰她，夸奖她，然后和她一起沆瀣一气的开始说奶奶的坏话，开始说爷爷的坏话，开始说老大家的坏话，开始说老三老四家的坏话，她就成功了，她就胜利了，可以不费一分钱的力气，就可以得到想要的口碑和利益，抬高自己贬低别人，这是多么划算的事情啊。凡事跟她有关系的人，不仅仅是爹妈，还有兄弟姐妹，还会败坏下一辈的名声，老大家的儿子多么的不仁义，小时候奶奶多么疼他，等长大了，却一点也不回报奶奶，仿佛在说，就是因为老大家的人品不好，所以才教育出来了这么一个不争气的儿子，这真是滑天下之大稽啊。爷爷奶奶的孙子孙女，外孙女外孙子中，该买东西的买东西，该去看忘的也都时常的去看望，相反的就是奎东家的好儿子，好闺女，可是一下也没有来看过，就好像他们的爹妈是从石头缝里蹦出来的一样，就像她说的一样，自己是从树上长的，加之她的儿子姑娘也是没有爷爷奶奶的人。

    4.泯灭人性

    为了将来有一个埋他的地方，奎东和她老婆终于算是同意去照顾痴呆的爷爷和行将就木的奶奶了。这个过程也是十分的曲折，仿佛是他们发了善心才同意去看望老人的，而不是因为责任才去的。这是多么伟大的付出啊，父母一天没有照顾过自己，自己还要大发慈悲的去照顾他们，这是多么光荣和值得大书特书的事情。

    当值班的第一天，那不负责任的态度可就体现的淋漓尽致了。当奎东的娘躺在床上的时候，因为身体的疼痛，不断的哀嚎，不想着去看看给她翻翻身，问候下到底是哪里不舒服，只是一味的坐着玩手机，耳朵就好像聋了一样，任由奶奶难受，哭天抢地的，躺在床上的不是自己的母亲，只是路边的一条野狗，一个野猫，死了活了和自己没有一点点的关系。你哭任你哭，我自独快活，我来了，就是我进到了作为子女的义务了，至于你是死了还是活了，都不管我的事情。到了点就走了，晚上也再也不会出现了，然后出门就要到处去说：“你看，我多么的孝啊，我爹妈对我这么的不公平，我还要伺候她，我多么的伟大啊。”

    除了出现过，坐下来，其他的一点用处都没有了。从来不做饭，从来不打扫卫生，躺在床上的奶奶生活无法自理，拉屎撒尿的，也不会去给换一个尿不湿。痴呆的爷爷，经常也是忘记上厕所，所有的屎尿的都粘在了身上，奎东和她的老婆，那就是看不见，听不见，不会去给爷爷换一条裤子，给奶奶换一个纸尿裤，就任由他们的身上粘满了屎尿，寒冷到骨子里。到了吃饭的点，看到家里有什么，有什么就给吃什么。看着别人买了点饼干，就用热水泡了泡，给爷爷吃了。有月饼，那就吃月饼，有面包那就吃面包，自己从来不会做一顿饭，哪怕出去买一个热乎的包子，那也是不可能的。奎东和她媳妇的做人哲学就是，我可以从我爹妈那里不断的往家里搬东西，但是要让他在自己的爹妈身上花一分钱，那是不可能的，万万不可能的。

    一家轮班值两天的班，奎东或者是他媳妇，总是中午来一趟，晚上就不来了，也不给换尿不湿，也不给翻身，两天之后，整个的尿不湿都已经湿透了，床垫子都已经湿的透透的了，奎东和她的媳妇，每次来交接班的时候，好像只是为了确认老人家有没有死了一样，如果死了，好赶紧收拾东西，分家产啊。我无法想象两天以来，奶奶经历了什么，没有东西吃，除了病痛，还要忍受着身子下面的潮湿和尿臭，等两天过去了，接下来的二姑，就要不断的给奶奶擦身子，还要把奶奶身子下面的床垫子拿下来去洗洗晒晒，老二家造的孽，总要别人给他们擦屁股，这是多么自私的一家人啊。老二家的媳妇这个样子，可是连奶奶的亲儿子，奎东也这个样子，甚至连去看自己的亲妈都不去，这是什么样的心肠呢？

    5.偷盗成性

    我在奶奶家装了一个摄像头，每一次她轮到她值班的时候，要做的一件事就是，先用一块布把摄像头遮起来，生怕别人看到她做的那些龌龊的事情。口口声声的说，要买一个录音笔放到家里，看谁拿了东西，越是喊的响的人，心里才是最心虚的人。每一次轮到她去值班的时候，看着奶奶家有什么，就不断的往自己家里搬。奶奶的侄子侄女，外甥外甥女，外孙外孙女还有孙子孙女，以及老大家的一些朋友，为了孝敬老人，都买了很多的东西。头一天送过去了，奶奶还一点没看到，第二天就被她顺回了家里，就像偷米的老鼠，看着奶奶家有什么好的东西，什么牛奶、面包，什么鸡蛋八宝粥火腿肠，桃酥，核桃露，凡事能看到的，能想到的，奎东和她媳妇就一个劲的偷，从来不想着这是为了给老人补身体用的，是别人的恩情，她完全当成了自己的东西。除了偷东西，明明是自己偷东西，还到处的说，你看看老大家整天去，都把东西搬光了，老三老四都拿了多少多少东西，什么事贼喊捉贼，这就是活生生的例子。

    无法理解的是，她自己没有亲戚朋友的来往，甚至自己也从不往奶奶家买东西的人，甚至连一个包子都没有买过的人，拿别人的东西的时候，竟然一点不脸红心跳，一点都不觉得羞愧。我也不能说不理解，就像我无法理解小偷的心情，无法理解地沟里的老鼠的行为是一个道理的。

    6.忘恩负义

    滴水之恩当涌泉相报，更何况是十月怀胎，辛辛苦苦的拉扯大了，还给取了媳妇，连这一点恩情都不念了，还能指望这样的人能对其他的人有感激之情吗？奎东和他的媳妇就能做到。不单单说这个养育之恩了，还有别人的帮助，似乎都是欠他们的。他从来不会想着他的兄弟曾经给他盖过房子，给他娶过媳妇，自己的妹妹把自己辛辛苦苦做衣服卖的钱都补贴给他的家用了，给他的孩子买了多少衣服，买了多少好吃的。他也从来不记得自己住院的时候，是几个兄弟姐妹帮忙才度过的难关，自从出院之后，这一切都好像没有发生一样。他也从来不记得自己什么都不会，是自己的叔兄帮他找了工作，让他才有了工作能赚钱养活家里。好像这一切都是没有发生的事情一样。

    这样的人何以能活在这个世界上呢？怪不得已经五十多岁了，竟然一个朋友都没有，从来都是我行我素的一个人，但凡帮助过他的人，好像也都不会再帮助第二回了吧。

    8.懦弱无能

    你无法想象一个男人可以懦弱到什么程度，看了很多的电视剧，当自己真的遇到了这样的一个人，才真的深刻的体会到，原来电视剧里面演的都不是假的，奎东就是那个不是男人的人。从来不在老婆面前说一个不字，即便是老婆骂他爹，骂他娘，自己丝毫不关心，丝毫不在乎。我记得比较经典的事情就是，当他的妹妹在和他老婆理论的时候，他妹妹对他说：“哥，你说句话啊，咱妈没生你养你吗？没给你盖房子娶媳妇吗？没给你看孩子吗？” 他老婆说：”没有，他妈没管他，他是树上长的，一点爹妈的好处都没有得到。“ 妹妹瞪红了眼睛看着他，他竟然还是一句话不说，许久之后，脸憋的通红，说到：”我不当家，我说了不算。“ 老婆就开始说：“是的，他不当家，说了不算。” 这是怎么样的窝囊性格啊，连说一句正儿八经的话都坐不到，这可真是齐天下之大闻啦啊。连自己是谁生的，谁养的，都不敢说，不敢做，一辈子就要活在一个耻辱的环境中，活着还有什么意思呢？

    所有的事情都要回家问老婆，仿佛自己就是一个傀儡一样，什么都是以老婆马首是瞻，无论是对的还是错的，没有自己的思想和决定，方法就被控制了一样，提线木偶的生活，活着还有什么意思呢？还不如早点死了算了。

    9.厚颜无耻

    我从未见过如此厚颜无耻之徒。奎东的媳妇亲妈死的时候，为了找人给撑场面，寻思着婆家不能不去人吧，于是托人来找了我们家，叫我妈去一趟，算是帮个人场吧。我妈不顾前嫌，两家毕竟已经多年没有交集了，现在既然人家有求与我妈，那我们也就不计较以前的事情了，就去了一趟。奎东的媳妇非但不感激，在葬礼上，竟然也不待见，我妈赶着给她说话，她竟然理都不理，仿佛不是她求着我妈去的，而是我妈不要脸的为了巴结她而死皮懒脸的跟上的。先不说我妈花了多少钱，但就这个礼遇，也总是不对的。如果早知道是这样的结果，当初就不要来求着我们去啊，干嘛还如此不要脸的来求着我们，最后还不以礼相待呢？

    10.造谣生事

    总结

    很多的东西，很多的东西，写不完，说不完，只有自己经历过才能深切体会，什么是一个无耻之徒，什么是一个让人恶心到想吐的人。济公有一集是讲的瘤子的故事，曾这么说：“善有善报，恶有恶报，不是不报，时候未到。人善福自来，心慈少病灾，恶生千百祸，阴险果悲哀。” 这个纷繁的世界就是缺少了一些公平，总是让坏人横行，让好人受苦。但也应该相信，这样的人总是不会有好下场的，从她教育的好儿子就可以看出来。

    isNaN 和 Number.isNaN 函数的区别？

    1.函数 isNaN 接收参数后，会尝试将这个参数转换为数值，任何不能被转换为数值的的值都会返回 true，因此非数字值传入也会返回 true ，会影响 NaN 的判断。

    2.函数 Number.isNaN 会首先判断传入参数是否为数字，如果是数字再继续判断是否为 NaN ，不会进行数据类型的转换，这种方法对于 NaN 的判断更为准确。

    为什么0.1+0.2 ! == 0.3，如何让其相等?

    一个直接的解决方法就是设置一个误差范围，通常称为“机器精度”。对JavaScript来说，这个值通常为2-52，在ES6中，提供了Number.EPSILON属性，而它的值就是2-52，只要判断0.1+0.2-0.3是否小于Number.EPSILON，如果小于，就可以判断为0.1+0.2 ===0.3。

    == 操作符的强制类型转换规则？

    1.首先会判断两者类型是否**相同，**相同的话就比较两者的大小。

    2.类型不相同的话，就会进行类型转换。

    3.会先判断是否在对比 null 和 undefined，是的话就会返回 true。

    4.判断两者类型是否为 string 和 number，是的话就会将字符串转换为 number。

    5.判断其中一方是否为 boolean，是的话就会把 boolean 转为 number 再进行判断。

    6.判断其中一方是否为 object 且另一方为 string、number 或者 symbol，是的话就会把 object 转为原始类型再进行判断。
