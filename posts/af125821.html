    人生经验之教育

    有时候确实很多的时候，都是挺让人费解的，无论你苦口婆心的怎么说，想要在短时间内能让一个人爱上学习，可真是太难了。

    理解他人

    理解万岁，体谅别人的幸苦。你可以不认同，但是一定要理解。我很少有记住的。

    不要无动于衷

    如果你连一个基本的想法都没有的话，那么说再多的话都没有用的。

    最好年华

    人生有许许多多次改变命运的机会，越早开始，成功的几率越大。不是说你过了高考，考不上人生就没有希望，就要去寻死觅活的，而是说，在你年少的时候，尽自己最大的努力，成功的可能性就会比老年的时候再努力。

    相信才有可能

    洗脑的前提就是要相信，首先要相信，才能慢慢的学会接受。首先要相信，无条件的相信，才有可能从心底里认同，进而做出改变。

    度过叛逆期

    通过几天的观察，其实对于14岁的孩子来说，真是。

    学会学习

    授人以鱼不如授人以渔，教会一个人学习，不是教会人某一道题，而是学会学习。

    改变从鼓励开始

    教育孩子的时候，我觉得更多的是鼓励，而不是从批评开始。

    持之以恒的力量

    只有长年累月的积累，才能有质的飞跃。

    榜样的力量

    给自己树立一个榜样，以后成为什么样的人。钱学森、钱伟长、钱三强。

    兴趣是最好的老师

    主动学习和被动学习的效果是不一样的。把自己喜欢的事情做成一份事业，是一种幸福的事情。

    小的时候应该培养自己的兴趣爱好，但是到了初中高中，就应该适当的放弃自己真大不喜欢的东西。人的精力时间毕竟有限的，不要一味的贪多，贪多嚼不烂，找自己真正爱好的一件最多两件事情，作为平常消遣的事情。

    吃的苦中苦，方为人上人

    天道酬勤，意思就是说老天爷都会帮助和奖励勤劳努力的人。

    自律和他律

    无论是少年、青年和老年，都有自己的欲望，能克服自己欲望的人，就是自律的人，无法克服自己的欲望就不能做到自律，在迈像成功的道路上，就是要借助他人的力量去规范自己的行为，最终达到自由的状态。就像为什么有法律有道德来约束人的行为规范一样，你做的过了，杀人放火就要偿命就要坐牢，道理也都是一样的。你现在控制不了自己对手机的欲望，那就让别人帮你控制，叫亲戚朋友帮忙，帮助你克服自己身上的缺点，这就是他律了。特别是青少年时期，因为心智不全，需要监护人来规范他的行为。

    谨慎交友

    见贤思齐，见不贤而自省也。

    OKI

    学会分享

    把自己学到的知识，能通过文字语言表达出来，那这个知识也就真正的属于你了。

    敬畏

    时常怀有敬畏之心，人就更加的能够约束自己的行为。

    早恋的引导

    isNaN 和 Number.isNaN 函数的区别？

    1.函数 isNaN 接收参数后，会尝试将这个参数转换为数值，任何不能被转换为数值的的值都会返回 true，因此非数字值传入也会返回 true ，会影响 NaN 的判断。

    2.函数 Number.isNaN 会首先判断传入参数是否为数字，如果是数字再继续判断是否为 NaN ，不会进行数据类型的转换，这种方法对于 NaN 的判断更为准确。

    为什么0.1+0.2 ! == 0.3，如何让其相等?

    一个直接的解决方法就是设置一个误差范围，通常称为“机器精度”。对JavaScript来说，这个值通常为2-52，在ES6中，提供了Number.EPSILON属性，而它的值就是2-52，只要判断0.1+0.2-0.3是否小于Number.EPSILON，如果小于，就可以判断为0.1+0.2 ===0.3。

    == 操作符的强制类型转换规则？

    1.首先会判断两者类型是否**相同，**相同的话就比较两者的大小。

    2.类型不相同的话，就会进行类型转换。

    3.会先判断是否在对比 null 和 undefined，是的话就会返回 true。

    4.判断两者类型是否为 string 和 number，是的话就会将字符串转换为 number。

    5.判断其中一方是否为 boolean，是的话就会把 boolean 转为 number 再进行判断。

    6.判断其中一方是否为 object 且另一方为 string、number 或者 symbol，是的话就会把 object 转为原始类型再进行判断。
