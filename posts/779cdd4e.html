    小米MIUI投屏到电脑

    标签： 无 分类： 未分类 创建时间：2019-06-13 00:53:26 更新时间：2025-04-28 14:37:41

    有时候，使用电脑操作手机还是很方便的。

    1.安装Vysor软件

    Vysor是一个连接手机和电脑的软件：vysor ，不仅有桌面端软件，还包括了Chrome插件，在谷歌应用商店中搜索并安装Vysor（需要科学上网）。

    2.打开MIUI10的开发者模式

    MIUI10打开开发者有点变态，百度经验上你一搜，可能并不能找到你要的内容，因为在设置->更多设置里面，并没有开发者选项。这个时候，先打开设置

    然后点击我的设备

    点击全部参数

    连续点击MIUI版本这一选项7次

    点击几次后，下方会提示你还有几次可以进入开发者模式

    回退到设置，更多设置选项

    这个时候，更多设置里面就出现了开发者选项

    3.启动USB调试和USB调试（安全设置）

    在更多设置的开发者选项中，打开USB调试、USB调试（安全设置）和USB安装选项。USB调试（安全设置）是为了能实现在电脑上操作手机而开启的，否则只能观看手机上的内容，而不是能操作手机。USB安装选项，是为了让Vysor能自动在手机上安装它自己的apk。

    4.将手机和电脑用数据线连接

    这一步没得说，找根数据线将手机和电脑连起来就好了。

    5.打开Vysor软件

    安装完谷歌的插件后，在win10开始菜单中的Chrome应用中，会有一个Vysor图标，点击即可打开。

    6.查找设备

    点击Find Devices设备

    7.连接设备

    如果Vysor在手机上安装了好了Vysor软件，这里就会出现一个Android设备，选中设备，点击选择即可连接。

    手机上的Vysor软件

    至此，电脑连接手机，就完成了。

    缺点：
    1.帧速率不是很高，图像有延时，快速操作会模糊失真，但是延时一会就清晰了。

    isNaN 和 Number.isNaN 函数的区别？

    1.函数 isNaN 接收参数后，会尝试将这个参数转换为数值，任何不能被转换为数值的的值都会返回 true，因此非数字值传入也会返回 true ，会影响 NaN 的判断。

    2.函数 Number.isNaN 会首先判断传入参数是否为数字，如果是数字再继续判断是否为 NaN ，不会进行数据类型的转换，这种方法对于 NaN 的判断更为准确。

    为什么0.1+0.2 ! == 0.3，如何让其相等?

    一个直接的解决方法就是设置一个误差范围，通常称为“机器精度”。对JavaScript来说，这个值通常为2-52，在ES6中，提供了Number.EPSILON属性，而它的值就是2-52，只要判断0.1+0.2-0.3是否小于Number.EPSILON，如果小于，就可以判断为0.1+0.2 ===0.3。

    == 操作符的强制类型转换规则？

    1.首先会判断两者类型是否**相同，**相同的话就比较两者的大小。

    2.类型不相同的话，就会进行类型转换。

    3.会先判断是否在对比 null 和 undefined，是的话就会返回 true。

    4.判断两者类型是否为 string 和 number，是的话就会将字符串转换为 number。

    5.判断其中一方是否为 boolean，是的话就会把 boolean 转为 number 再进行判断。

    6.判断其中一方是否为 object 且另一方为 string、number 或者 symbol，是的话就会把 object 转为原始类型再进行判断。
