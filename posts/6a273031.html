    人生经验之基层腐败

    标签： 随笔 分类： 未分类 创建时间：2021-07-25 02:21:49 更新时间：2025-04-28 14:37:41

    最近回农村老家，遇到一个奇葩的事情，这让我开始回想起来，这许许多多年，一个山东小小的农村及其村干部的无耻工作。

    1.清理房前屋后的菜园子

    2021年07月25日，最近不是在创建文明城市吗，于是村里的广播上就开始广播了，“最近创建文明城市比较紧，截止到今天下午，各位村民，要把自己房前屋后种的黄瓜、茄子、辣椒等蔬菜都处理掉，如果不处理掉，明天村里就会组织人，帮助清理。” 我不知道这个政策是不是县里、市里还是省里做的决定，我其实就是有点不太理解，为什么放着更大的，更多的，更重要的工作不做，非得要在这些细枝末节上进行内卷呢？难道那一个个的工厂不应该进行管理吗？城市马路上的垃圾不应该有人打扫吗？为什么这么一个偏远的小农村，要为了进行表面的工作，而要农民把自己屋前的菜地都要除掉。

    2.村委会被悄悄的卖掉了

    村里的大队，也就是村委会，用了很多的年的房子，有三个篮球场那么大，理论上，应该是村民的公共财产，属于公家的资产。在我小的时候，门口还只是一面墙，门很大，后来，把墙拆了，加上路上的一点，被盖起了房子，卖给了我小学同学的爸爸，于是这一家人，就在村委会门口住了下了，这片地也成了私人的地，但是这个时候，里面的院子和院子里面的房子，还是属于村委会的，村委会也还是在里面办公。现在，连最后的一点院子和院子里的房子，最近也被村委会卖给了个人。本来是在村子的中心地带，方便四面八方的村民集中到这里进行开会，选举，办理各项事务等，卖掉之后，村委会也搬到了村子最东边，和村子相隔很远的一个偏僻的农家院里，在几年前，还是一片稻田的地方，从此，要开什么会，办什么事，都要走很远的路去那里了，新的村委，周围根本就没有什么人住，都是种粮食的地。

    我也无法理解这样做的目的，关键是卖地也没有通知村民，也没有举行什么村民大会，进行表决什么的，都是村干部自己做主了，把原本属于村民的财产，属于公家的财产，变成了他们自己的东西，说卖就卖了，说迁就迁了，我也不知道这样做，上面的领导知道，难道镇上的领导，区里的领导，都是瞎子吗？都是聋子吗？难道这么大的事情，真是合理合法合规的吗？村里人为什么不向上反应呢？可是反应了，有什么好处呢？假如这件事情，上面就是已经知道了，那自然而然就是不给处理，甚至还会把反应的人抓起来，假如不知道，查起来，顶多也就是息事宁人罢了，难道还会有什么别的动作吗？这让我想起了另外一件事。

    3.村支书被撤职了变成了村长

    这件事也是挺有意思的，就是有一个叫刘学信的村支书，因为贪污，被上面免去了村支书的位子，但是奇怪的地方，就是并没有从村委会除名，不干村支书了，还是村长，继续贪污受贿，继续自己说了算。

    isNaN 和 Number.isNaN 函数的区别？

    1.函数 isNaN 接收参数后，会尝试将这个参数转换为数值，任何不能被转换为数值的的值都会返回 true，因此非数字值传入也会返回 true ，会影响 NaN 的判断。

    2.函数 Number.isNaN 会首先判断传入参数是否为数字，如果是数字再继续判断是否为 NaN ，不会进行数据类型的转换，这种方法对于 NaN 的判断更为准确。

    为什么0.1+0.2 ! == 0.3，如何让其相等?

    一个直接的解决方法就是设置一个误差范围，通常称为“机器精度”。对JavaScript来说，这个值通常为2-52，在ES6中，提供了Number.EPSILON属性，而它的值就是2-52，只要判断0.1+0.2-0.3是否小于Number.EPSILON，如果小于，就可以判断为0.1+0.2 ===0.3。

    == 操作符的强制类型转换规则？

    1.首先会判断两者类型是否**相同，**相同的话就比较两者的大小。

    2.类型不相同的话，就会进行类型转换。

    3.会先判断是否在对比 null 和 undefined，是的话就会返回 true。

    4.判断两者类型是否为 string 和 number，是的话就会将字符串转换为 number。

    5.判断其中一方是否为 boolean，是的话就会把 boolean 转为 number 再进行判断。

    6.判断其中一方是否为 object 且另一方为 string、number 或者 symbol，是的话就会把 object 转为原始类型再进行判断。
