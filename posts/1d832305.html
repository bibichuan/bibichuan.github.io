    人生经验之奋斗

    标签： 无 分类： 未分类 创建时间：2023-05-22 07:58:17 更新时间：2025-04-28 14:37:41

    最近总是有视频给我推送，就是年轻人不应该吃苦。年轻人到底要不要吃苦，该不该吃苦？是否真的就应该选择躺平呢？996到底是不是福报？在我看来，我觉得这个应该从客观公正的角度去分析，年轻人应该吃苦，但是不能996。奋斗和躺平，其实两者并不矛盾。作为一个顶级的社畜，我反对996，但是我也反对不思进取。我自己虽然确实没有什么志向，没有什么努力，但是我鄙视我自己。

    我觉得很多的视频也好，UP也好，说的东西过分的片面了，我不并不敢苟同。“年轻的时候就应该吃苦” 这句话对吗？这句话有可以批评和指摘的地方吗？所谓的吃苦，并不是让你去吃糠咽菜，虽然很多的时候家长总是在说，自己以前怎么怎么苦，怎么怎么累，怎么怎么吃不上饭，但是现在的吃苦的含义应该是一种奋斗的精神。我们过着优裕的生活条件，也还是要努力的做出成绩，要在自己喜欢的领域，喜欢的事情上做出成绩，想要出成绩，那么自然而然的就要吃苦，就要付出汗水。从这个角度去理解年轻的时候，应该吃苦，是不是就正确了呢？

    “年轻人就应该吃苦。” 第一个角度就是说，要趁年轻的时候吃苦。不趁着年轻时候自己受点累，受点罪，难道等老了再去吗？上高中的时候，要起早贪黑的学习，付出时间，付出汗水才能考上好的学校。在大学的时候，只有耐得住寂寞，在图书馆，在教室，在实验室努力的啃课本，读书，才能有一个好的成绩，才能顺利毕业，找到好的工作。考研的时候，就像很多的考研学子，也都是起早贪黑的跑道自习室，如果你不吃苦，就像我一样，30岁了，还在梦想着回到校园，一辈子都是遗憾。

    “年轻人就应该吃苦。” 第二个角度就是说，要吃苦，要吃什么苦。吃不好的饭，过破衣烂衫的生活，这也是一种苦，但不是新的年轻人吃的苦。资本家口中的要奋斗，要996或者007，也不是应该提倡的文化。我觉得应该努力的地方，就是要花费巨大的时间精力在自己喜欢的事情上，在不断的学习和提升自己的能力上。无论是科研能力，无论是演讲能力，还是音乐、美术、写作能力，每一样都会在精神和体力上让自己吃的苦中苦，方为人上人。考研的日子苦吗？苦。每天都要起早贪黑的去占座，一坐就是七八个甚至是十一二个小时，每天要读很多书，看很多的课，没有一点娱乐的时间，这不就是苦吗？这样的苦到底要不要吃，我觉得应该吃，经历过翻阅过高山的人，才能知道山顶的风景是如此的美好。

    还有一个最近比较尝说的词就是 “卷”，工作卷、学习卷、旅游卷，各种各样形式的卷。为什么会造成这样的现象呢？因为世界是不公平的，因为中国的人多，因为绝大多数人希望自己过的比别人幸福，所以 “我要悄悄的学习，然后惊艳所有人。” 这句话并不是没有道理的。你不卷，别人也在卷，别人不卷，最后就是被淘汰，老话说得好，落后就要挨打，想想清末的时候，多么惨痛的代价啊，没有坚船利炮，只能割地赔款，这就是现实。

    所以我说，年轻人该吃苦，该奋斗的时候奋斗，该学习的时候学习，该努力的时候就应该努力，但是也要用理智的态度去看待这份努力，向着能提升自己，升华自己的方向吃苦，而不是重复的劳动，艰苦的生活。

    isNaN 和 Number.isNaN 函数的区别？

    1.函数 isNaN 接收参数后，会尝试将这个参数转换为数值，任何不能被转换为数值的的值都会返回 true，因此非数字值传入也会返回 true ，会影响 NaN 的判断。

    2.函数 Number.isNaN 会首先判断传入参数是否为数字，如果是数字再继续判断是否为 NaN ，不会进行数据类型的转换，这种方法对于 NaN 的判断更为准确。

    为什么0.1+0.2 ! == 0.3，如何让其相等?

    一个直接的解决方法就是设置一个误差范围，通常称为“机器精度”。对JavaScript来说，这个值通常为2-52，在ES6中，提供了Number.EPSILON属性，而它的值就是2-52，只要判断0.1+0.2-0.3是否小于Number.EPSILON，如果小于，就可以判断为0.1+0.2 ===0.3。

    == 操作符的强制类型转换规则？

    1.首先会判断两者类型是否**相同，**相同的话就比较两者的大小。

    2.类型不相同的话，就会进行类型转换。

    3.会先判断是否在对比 null 和 undefined，是的话就会返回 true。

    4.判断两者类型是否为 string 和 number，是的话就会将字符串转换为 number。

    5.判断其中一方是否为 boolean，是的话就会把 boolean 转为 number 再进行判断。

    6.判断其中一方是否为 object 且另一方为 string、number 或者 symbol，是的话就会把 object 转为原始类型再进行判断。
