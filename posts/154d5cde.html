    人生经验之杭州租房

    标签： 租房 分类： 随笔 创建时间：2019-07-29 06:04:39 更新时间：2025-04-28 14:37:41

    前言

    在杭州已经生活了将近8年了，早些年的时候，从大学毕业，到进了单位，单位提供了宿舍，后来搬迁之后，单位又帮忙租了房子，充当宿舍，前前后后共计有四年的时间，不用自己出房租，不用自己找房子，现在想想还是挺幸福的。

    到了自己辞职的时候，那时候第一次出来找房子，因为本打算和自己的朋友一起住，于是就找了一个整租的，跟着我爱我家的工作人员走了走了很多家，最后找了一个两室一厅的房子，老破小，签约也是和我爱我家签的。当时不追求什么生活品质，为了省钱，一度将主卧转租出去了，自己住了一个仅有六七平的次卧。

    次卧住了一年，房租涨了，不想再住了，于是就再次的找房子，那个时候，蛋壳公寓还在，还没有暴雷。蛋壳也找了，附近的几个小区也找了，最后机缘巧合之下，通过小区的公告栏，竟然找到了一个带阳台的主卧，房间很大，加上阳台，足足有20平，厨房也比较大。唯一的缺点就是没有电梯的六楼，楼上也住了人，有一个阁楼，有一个露台，可以晒衣服，挺好的。除了第一搬家比较费劲之外，平常上下班也没有觉得有多么难。

    在这个蓝天城市花园住了有两年的样子，期间因为种种原因，也曾经通过中介找过两次房子，不过两次房子都没有足成，只是因为自己个人的原因，所以还就一直住着。

    到了2022年的时候，工作换了，不得不找新的住处了，恰好有一个朋友空了一件loft，一个月1900，后来涨到了一个月2500，如果一个住挺舒服的，两个人住的也还是可以的。这次租房也没有费什么劲，反正就是直接和朋友谈好了，然后直接就去搬家住进去了。

    后来因为种种原因，又要找房子了，这次还是先去了贝壳上找房子，但是贝壳的其他家的中介基本上都没有了，只剩下了自如的房子了，但是自如的房子除了贵，其他的也基本上没有什么问题。我看了好几天，还是找了一个看起来还可以的房子，联系了自如的管家，他们基本上的套路都是，你看上的房子他说没有了，然后带着你去其他的房子，这真是一个挺恶心的讨论。自如的房子如果带阳台的，基本上都是1800以上了，如果想要三室的，也基本上没有，都是一些隔断房，一个房子里面有五六个房间。除了自如，我还去了一些回迁房，只要你去小区门口那种大爷大妈比较多的地方，而且有公告栏的，基本上都贴着房子出租，你只要一过去，大伙就会围过来，然后问你要不要租房子。我还是有时间，于是就跟着他们挨家挨户的去看，少说也看了七八家吧。基本上都是独卫，但是有一个特别的小的厨房，就像一张桌子一样，上面装了一个油烟机，因为都是隔断房，所以基本上都是很小的房子，大致上就是1500到1800，有阳台的就是1900块钱了，这样的小区我倒是看了好几个了，在秀月家园看了，在万家花苑看了好几个，还有那种只有一张床的，连晒衣服的地方都没有的，还需要1200块钱，真是令人感到不快。还有豆瓣租房，但是豆瓣租房都是中介，基本上也都是有中介费的。我还在一个小程序叫 “唯心所寓” 的上面看了几个，这种只能看缘分了，要是运气好的，可以找到真的是转租的，运气不好，那就可能人家连回复都不会回复你了。

    房地产的行情不景气，可是租房的市场还是那么的火爆，都是那种单间的房子，最后还能租1500到1800，我老家的市区的房子，110平的房子，也才租了1200块钱，真是不公平啊。

    2019年7月29日

    最近已经提出离职报告了，工作很久，一直都是住的单位给租的房子，虽然环境差，一个房间住的人多，以前住在教工路那里，一个宿舍住五六个人，房子多大？30平撑死了吧，而且房子很老，卫生间在外面，门都关不上的那种，把手也没了，只能用布塞住，幸亏是男生，也不在乎，当然同层里面也有其他的宿舍，甚至还有女生，别的房间的卫生间都在里面的，我就算洗完澡穿着裤头从洗刷间回宿舍也不用担心（当然宿舍和洗刷间挨着）。后来单位搬到了新基地，在阿里巴巴（西溪园区）附近，单位也很人性的在附近租了一幢别墅，将其改造成了宿舍，一个房间住两到三个人那种，环境改善了许多，厕所出来一楼和三楼是公共的以外，二楼的主卧和次卧以及半地下室的一间卧室都有独立的卫生间。我很幸运的分到了一个二楼次卧的一个床位，毕竟能20分种走到单位，已经很不错了，而且由于房间少，也不是所有宿舍的员工都住进来了。一个房间住的人虽多，但是不用担心各种物业费，水电费等，东西坏了，报给单位，单位也会找师傅来修。对于我这种杭漂来说，已经很不错了。

    如果杭州有房子的话，其实呆在这个单位，努（hun）力(chi)奋(deng)斗(si),争取能成为领导，为国献身，也挺不错的。可是，对于如今的经济形式和房价形式，都不动就上两百万的房价，即便我砸锅卖铁凑齐了首付，也付不起每个月一万的房贷。无奈之下，只能离开，离开舒适圈，有万分之一的希望能获得更好，不离开，连一点希望都没有。我不能祈求国家可以真的控制房价（大庇天下寒士俱欢颜，到那时候，我可能已经入土了吧），只能学会自己改变。
    切入正题吧，改变的第一要做的，就是租房子，离开了单位，不能赖着人家的地盘不走吧。租房子，真的是挺头疼的事，自如APP、58同城、赶集网，我都看过了，连豆瓣小组也不放过，看来看去，挑来挑去，各种方式避免遇见中介，最后，还是中介的帖子最多。中介的身影无处不在，中介的套路无所不能。看看这一个个的小组，一个个的帖子，满满的都是 “哥，房子看一下。” “哥，你想要什么样的价位。哥，你考虑的怎么样了。哥。。。”

    无论推销也罢，中介也罢（每一个向你推销产品的人，都比你有钱），大家都是为了混口饭吃，我理解，可是最怕的就是不能遵守职业操守和道德底线，将黑的说成白的，将白的说成黑的。好不容易看到了几个好的房子，价格也还能接受，一加微信，介绍里都明说了“租房找他”，不是中介又是什么呢？都没有想加的欲望了。

    我也试过在自如上查看房子的信息，可是那里的房子都很贵，不贵的话，也是四室五室，房间特别小，只够放一张床的，住了这么多人，上个厕所都是很麻烦。一直看，一直看，都连续看了一个星期了，还是没有找到位置合适（考虑到上班方便），价格便宜（主要是没钱）的房子。路漫漫其修远，吾将上下而求索。其实，我还是比较担心的，在租房的过程中，费时费力还得不到想要的结果。

    2019年7月30日

    听朋友介绍，上了杭州19楼论坛，找了几个帖子，确实挺吸引人的，结果，一问价，一加微信，都是中介。明明是中介，却硬要把自己写成房东，也说明了做人不够诚信，还真不如直接写自己是中介呢，明码标价，就是要中介费，我还能够接受。租个房子，是真难，真的难。

    2019年8月1日

    在这个中介满天飞的社会大背景下，可能很多的房东也不会上豆瓣啊，上自如之类的租房平台吧，把房子挂给中介，自己只收租金就好了，反正中介费又不是自己出，这又有什么关系呢？所以，我觉得要想真的在豆瓣小组，杭州19楼这种论坛性质的地方，找个靠谱的房源，真的是比登天还难，除非是那种自己租了房子，没到期，还想收回成本的，转租的情况，一般都是自己发帖，不会找中介。要想找个好点的房子，除了看论坛，还是要乖乖的找中介，论坛不是找不到，万一恰好有人想出租房子呢？万一有人恰好想转租房子呢？万一恰好你看到了这套房子呢？

    最后，我还是选择了找中介。昨天晚上联系了我爱我家的中介，看了有六套房子，大多是老破小的小区，有两间房子还可以，不过和网上挂出来的房价相差甚远，按他们的意思是第一个月优惠，网上报价是2800，然后中介说，第一个月按2800，以后每个月按3800算。当时我就不高兴了，甚至有点被欺骗的感觉。整个过程都是不开心的亚子。然后他们又说，还有几套房子，可以带着看看，既然来都来了，那还能怎么办呢？跟着逛了几个地方，感觉他们一个又一个的人来，问:”这个你带看了吗，那个你带看了吗。。。”感觉租房火爆，各种人都在看房子，但是我一个看房子的都没遇到。不知道是不是假象，还是真又这么多人看房。第一家总共看了四套，一套刚装修，感觉味道很重，不合适，一套第一层，进去就是一股子发霉的味道，倒是价格便宜，还有一套把客厅放了一张床，当两居室出租，是真的不方便。另外也是我爱我家的中介，带着看了另外的两套房子，都还可以。一套次卧没有风扇，也没有燃气啥的，对于我使用电磁炉的来说，也无所谓，主要房间干净就好了。

    最后敲定了没有空调，没有燃气的那个房子。挂牌价为2700，中介说可以帮忙谈，我说，那你看2400可以吗？谈话如下：

    怎么感觉有点熟悉的套路的味道。最后下午的时候，她给我打了两个电话，第一个电话说2550可以吗？我说，不行，本来多五十块也不算什么，但是能少不能多啊。她说，那在帮我谈谈。我说行吧，那你去谈吧。（我有点傻）
    快下班的时候，又给我电话，说，房东说了，可以的，2500一个月，我给他打了好几个电话才行的。（怎么感觉谈个价格那么难呢？）还问我今天晚上可以签合同吗？不签合同，先交定金也行。“容我考虑一下”。

    。。。。

    经过长时间的纠结和思考，最后我妥协了，谁叫我是租房子的呢。中介费（月租的一半）要交就交呗，房子破点就破点呗，租金这么高就这么高呗，谁让租房这么辛苦呢，唉，没有人认识就是不好啊，没有房子就是不好啊。

    最后，以自己的失败告终，最终还是找了中介，最终还是付了中介费，最终交了1250的保证金，最终….明天签合同。(简直太快有没有？都让我来不及思考了，就这么被带到坑里了。)

    钱都怎么没的？

    2019年8月2号

    今天晚上就要签租房合同了，压1.5付3，中介费1250元，第一次就要交12500元人民币（一听这么多钱，就肉疼，辛辛苦苦的干了那么长时间，最后啥都不剩，都给别人了，穷人越来越穷，而地主却富的流油，啥都不干，不用劳动，就可以靠房租过日子，这是什么样的社会主义啊。），祈祷自己遇见一个好的房东，希望我爱我家这种公司不要欺骗消费者（话说，即便遇见了不幸，自己也没处说理啊）。网上搜了一下，签合同的注意事项，怎么样不被骗之类的：

    1.确定房屋产权。应当通过各种途经，房产证啊、户口本啊、身份证原件等，明确签合同的人确实是房东，而不是其他的什么各种人。

    2.查看水电气表网费，确保上一次的费用都结算清楚。

    3.写明房屋内附属设施情况。冰箱、电视、洗衣机等是不是能用，坏了怎么修，谁修等。要对房屋维修及费用问题做出约定。

    4.明确合同日期及房租到期时间，交租方式。

    5.写明一方违约，终止合同如何结算费用。

    6.明确房屋位置、间数、面积、质量、租赁期限、租金及支付期限与方式等。

    7.要明确房屋是否在法律允许的范围内出租，是否是查封住宅之类的。（这个好像不太好了解）

    我觉得知乎上有些人说的很对，租客其实是个弱势群体，对于房东来说，反正押金是在他们手上，房子也是他们的，你不租，自然有别人租（这一点，在和中介的沟通中，可以感受到，你觉得价格高，是吗，那不好意思，你不租，我组个别人）。房子不愁租，押金也在手，天下还有谁能奈何的了他们呢？所以他们有可能无理克扣押金，挑各种毛病，各种刺，最后呢，租客啥都不剩，辛辛苦苦一个月的工资，就啥都不剩了，白出了一个月的劳动力。还能指望国家吗？何时能让买不起房的人，住的起呢？何时能让有好几套房子的人，能少一点呢…扯远了。

    上了贼船，今天正好有事，就没去签合同，想着明天去，沟通如下，满满的都是不情愿呢。（或许我多虑了）经过艰难的协商，最后还是选择了明天去签合同。

    2019年8月3日

    今天肉疼的花了12521元人民币（三个月租金7500元，押金3750元，中介费1250元，物业费三个月21块钱），终于把房子租来了。上午十一点签合同，明明就没能见到房东，直接去的我爱我家的线下店面进行的签约，签约，也主要就是我爱我家的工作人员把合同打印出来，你去签字就好了，整个过程不到二十分钟就解决了，交钱，签字，走人，就是这么个流程。我爱我家会有个管家，在房子里等你，然后带你去抄电表的度数，水表的度数，水表在哪里我倒是没看见，管家说，我上个月已经结清了，而且还多付了水费。姑且就相信他吧。交接完成之后，她拿着我的签字单走了，对我说，一个月内，家具家电都是免费维修的，一个月后，易耗品（水龙头、灯泡啊）之类的维修是要收费的，我在他们的公众号上保修了，还真是，下午救过来了，虽然该修的没修好，只是把花洒给安上了，像进门的灯泡、进门的厨房的水池堵塞漏水、还有次卧的纱窗坏掉了，主卧里面的床头柜，还有电视机柜都是坏的，以及主卧的灯罩坏掉了，都没有给修好，算了，也还是来日方长，等下个星期真正的搬过去了，再让他们来修吧。

    好了，接下来，就是打扫卫生了，把别人剩下的东西，统统都搬走，扔掉，然后把该抹的，不该抹的，统统都擦了一遍，和同学，以及女朋友，一起忙碌了大半个下午，总算是告一段落，打到回府，要先做一号线到凤起路，再转二号线到古翠路，再上130路公交车到五常公交站，最后顺着荆长大道走路1.5公里，十几二十分钟到达香洲里小区（老单位给租的别墅中的一个小卧室）。这一路下来，要花上一个半到两个小时，更是让人想吐血。

    忘记拍照上传了，改天补上。

    整个租房过程下来，除了前一个星期挑花了眼，也找不到合适的，没有去看房，从开始看房到签订合同，前后不过三四天的时间，一方面自己想尽快的找到房子，能定下来，下个星期准备离开这里去新单位上班，另一方面自己也没有什么经验，对于网络上铺天盖地的中介信息来说，我不会辩驳，既然都是中介的天下，那就干脆找个靠谱点的中介好了，最后选择了我爱我家，虽然花了一笔不小的中介费，但起码有些保障吧，比如说，房子的维修问题。有得有失吧。

    房子怎么说最后也是解决了，剩下的就是等着老东家给我解除劳动合同了，上个星期去综合办问了一下，总是推脱领导在忙，要等下个星期。新东家在周二的时候还给我打过电话，我以为周五会再次打电话问辞职进度，却没有打来，我竟然慌了，生怕在我租完房子之后，生意黄了，因为新东家一直催我尽快入职，最后一次，我竟然说，要三个月（也不是我胡说，领导就是这么给我说的啊），可能新东家生气了，也不得而知呢。

    最后我要吐槽一下这万恶的房地产市场，真的是让人心疼，辛辛苦苦用劳动换来的工资，最后最后，却成了资本家收的租金了。

    2019年8月4日

    昨天忘记说了，房东一般把房子全权委托给中介机构，所有事情都是中介代办，房东只负责提供最低的租金要求，然后啥都不用管，等着收钱就好了。物业，管理，打扫，收租，一切都不需要操心，甚至中介费都不用交，一般都是租客给出了。昨天问了一下，如果没到期解约的话，要收半个月的房租作为转租费。如果到期了续签，要提前一个月，由中介和房东重新谈过价格，确定合不合适，然后缴纳半个月左右的手续费，也就是新的一笔中介费，才能继续租住。

    2019年8月18号

    租了房子之后，一个多星期没来住。这个星期刚搬过来，住了一个星期，一直没有上传照片，今天才抽空晚上拍了一下，我租的两室一厅。房价很小，很乱。
    门一打开，是一个厨房，我在客厅拍的厨房。

    与厨房连着的是一个客厅。

    不同的角度拍摄的客厅。

    与厨房并排的是卫生间

    从厨房进入客厅，右手边是次卧，前面是主卧

    isNaN 和 Number.isNaN 函数的区别？

    1.函数 isNaN 接收参数后，会尝试将这个参数转换为数值，任何不能被转换为数值的的值都会返回 true，因此非数字值传入也会返回 true ，会影响 NaN 的判断。

    2.函数 Number.isNaN 会首先判断传入参数是否为数字，如果是数字再继续判断是否为 NaN ，不会进行数据类型的转换，这种方法对于 NaN 的判断更为准确。

    为什么0.1+0.2 ! == 0.3，如何让其相等?

    == 操作符的强制类型转换规则？

    1.首先会判断两者类型是否**相同，**相同的话就比较两者的大小。

    2.类型不相同的话，就会进行类型转换。

    3.会先判断是否在对比 null 和 undefined，是的话就会返回 true。

    4.判断两者类型是否为 string 和 number，是的话就会将字符串转换为 number。

    5.判断其中一方是否为 boolean，是的话就会把 boolean 转为 number 再进行判断。

    6.判断其中一方是否为 object 且另一方为 string、number 或者 symbol，是的话就会把 object 转为原始类型再进行判断。
