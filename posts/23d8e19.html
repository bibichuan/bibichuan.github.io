    人生经验之人生规划

    标签： 无 分类： 随笔 创建时间：2019-08-06 08:44:33 更新时间：2025-04-28 14:37:41

    2025年02月13日 星期四 晴

    没想到我从27、28岁，一直到了现在的33岁，还真是快啊，快的都让我心疼了，还有一年多就要35岁了，我现在年薪40万，到那时候，我再回来看这篇文章，我会怎样呢？还会一直这个样子吗？还会有工作吗？还会有活干吗？

    推荐一些文章:
    1.谈谈35岁危机 35岁后的五种出路:1.成为管理者或技术专家;2.一直在基层做老员工;3.考编制进体制;4.创业做个小生意;5.转行。如何应对35岁的危机:1.选择节奏稍慢、实力较强的公司;2.要注意学习的方法和方向;3.保持与直属上级的信任关系;4.合理设计家庭财务计划，避免高额负债;5.养成良好的生活习惯，积极维护身体健康;
    2.知乎高赞：35岁失业的程序员，都去了哪儿？ 开拓自己的戏路，这便是自己应该做的。
    3.10 年程序员的告诫，写的太真实了 如果我发现自己是公司里面最厉害的工程师，那就该离开了。

    2021年03月14日 星期日 晴

    其实我总有大把大把的时间用来学习，可是我最后都白白的浪费在了无聊的动漫和电视电影上面了，最后落得个一无是处，这又是何苦呢？身体身体垮了，失利视力不行了，最后连足以谋生的技能也荒废了。这又是何苦呢？我什么都不会做，什么都不会玩，就是在宿舍待了一天，睡了一觉，看了一天的电视，无聊，无聊，无聊到头顶。当我回想起自己这三十多年的奋斗，什么奋斗啊，其实根本就从来没有说的出口的奋斗，没有考试的拼搏，没有工作的拼命，更没有值得一说的恋爱故事，这一辈子，就是这么平平淡淡的浑浑噩噩的过着日子，过着不疼不痒的日子，我会花大量的时间在看动漫上，会花大量的时间在看色情上，会花大量的时间在游戏上，却从没有花大量的时间在学习上，如果我把浪费掉的时间都充分的利用起来，获取结果已经不可同日而语了，或许我就不会考不上研究生了，或许，我现在可以有更高的资本去谈梦想，谈理想。其实啊，不是35岁会有危机，只是因为到了35岁，已经没有办法再继续浪费时间了，没有办法再继续无忧无虑的浪费时间了，也就没有多少时间可以用来浪费了。

    2021年02月16日 星期二 晴

    当我爱情濒临崩溃的时候，我再次思考起自己的人生。如今，我已经29岁了，虚岁已经30岁了。30而立，回想自己的前半生，看看自己现在的状态，还是一无所有。没有自己的房子，没有相当的存款，工作也是糟糕的一塌糊涂(在一个看不到前途的小公司里，陪着老板一起做着发财的美梦)，连最后的爱情，也被自己糟蹋的一点不剩。没有了爱情，我以后的路该走向那里，是继续留在杭州，还是去南京，或者济南重新开始自己的新的生活呢？曾经，因为我的对象还是浙江的，我还有理由以及借口，必须在杭州留下来，如今这个借口也没有了，我还有在杭州留下来的必要吗？未来啊，我该如何去选择？

    2020年10月17日 星期六 阴

    早上醒来，还是觉得自己真是一点也没有上进心，虽然每天都借口自己工作很忙，时间很少，但是实际上自己就是逃避，逃避去考试，逃避学习新的东西。一直在梦想着能在30岁以前考上研究生，可是，今年已经28了，却还是没有什么动静。发财，买房子，拖一天，房价就高一天，自己就要多奋斗30年，可能等自己老了的时候，死了的时候，还是还不起房贷。

    2020年04年28日 星期二 多云

    昨天听到了朋友去了上海Esri公司，工资很高。我又一次想到了，我们同时来到杭州，共事一个公司长达四年,如今要各奔东西了，以后杭州，又少了一个可以依靠的朋友啊，我觉得挺伤心的。可是我也会恭喜他，替他高兴，也贺喜他。祝愿他，平步青云，以后前程似锦。发达的时候，不要忘记曾经的兄弟。其实即便他有能力把我招进去，我可能也不会离开杭州去上海了。毕竟还有女朋友在杭州，而女朋友的工作，也不是说换就能换的。
    我也思考起自己的人生来了。为什么我第一次跳槽，就跳到了一个小公司而不能面试上大公司呢？为什么我要996,而别人只需要上五天班，就可以拿到我要拼命才能拿到的工资。会不会是因为我没有能拿的出手的项目，朋友还做过天地图浙江，而我，似乎都是在小项目上挣扎，没有能说的出口的项目。其实我也很想能有拿的出手的项目，可是，机缘巧合，刚进公司的时候，我就被赋予了一个救火员的职责，哪里需要就去哪里，什么时候有项目了，就临时的抽调，然后就去开发这个项目，都是一些小的不能再小的项目了。什么这个旅游系统啦，那个旅游系统啦，什么内部巡检系统啦，什么征收系统啦，最后留下的，还在更新换代的项目，已经没有了。跳槽已经有差不多一年了，做的还是一些打杂的活，都是一些示例项目，虽然也算一个小的团队管理者，面试了二三十个人，却也没有什么能学到的东西，因为没有可以借鉴的人，也不会知道自己做的到底是不是对的，错的，到底该怎么去改，怎么去修改。
    今年，我已经28周岁，29虚岁了，马上到了三十而立的年纪。我再次审视起自己的人生来了。

    (1) 接下来的人生中，希望能尽快的稳定自己的家庭，能完成时代交给的任务。

    (2) 接下去的人生中，如果有机会无论如何还是要去大公司去历练一下，和优秀的人一起共事，和优秀的团队一起共事，那我也就知道该如何做到优秀了，而不是自说自话，自以为是，自认为自己做的都是对的。

    (3) 如果不能再次进入校园，再次成为一名学生，没有一个研究生的学历，人生是不是非常的遗憾呢？我将全力以赴，将自己的学历提升，让自己在时代的浪潮中成为佼佼者。学历不代表能力，但是有了学历，一定比没有学历的的情况下，更加的优秀了。

    2020年01月01日 星期三

    就在浑浑噩噩中度过了一个又一个的春秋。

    最近看了一篇文章，将的就是努力与人性的话题，不是什么有名的文章，确实一个不错的思路，文章摘录如下：

    你崇尚旅行，在朋友圈里发了好多次“身体和灵魂，总有一个在路上”，可很久之后才发现，身体和灵魂一直在床上。
你酷爱自由，在微博里写着“生活不只眼前的苟且，还有诗和远方”，可一想到去远方的路途遥远，你最终还是选择了得过且过。
你喜欢稳定，处处流露着一种“面朝大海，春暖花开”的安逸，可现实慢慢让你明白，你所谓的稳定，其实是在稳定地穷着。
你说要早睡早起，给自己准备了很细致的计划表，还工工整整地贴在床头，可躺下之后发现根本睡不着，于是翻翻手机，看看朋友圈，一不小心就到了凌晨三点......
你说要多读书，努力充电，以此来升职加薪，可是一本本的专业书买回来，翻来覆去还是停在了第一页，顶多是拍个照片，配几句鸡汤文字，以示自己在努力学习。
你说要瘦成一道闪电，可一看到好吃的心里就小鹿乱撞，于是大吃了一顿，顺便还发了一个朋友圈，配图文字是“我要好好爱自己”。

    2019年08月06日 星期二

    在我写下这片文章的时候，是2019年8月6号，我时年27周岁，虚岁28岁，马上到了而立之年，除了一个稳定的工作，竟一事无成（无车无房），如今，这份稳定的带编制的工作我也要放弃了，只是为了出去能多挣点钱，在这个疯狂的年代在杭州买一套属于自己的小房子。其实，如果有的选的话，我可能真的会选择在事业单位里混吃等死，收入不要求多高，能养的起家，满足温饱即可，偶尔可以改善下伙食就已经心满意足了。然而理想很丰满，现实很骨感，赤裸裸的现实，逼着和我一样的年轻人踏上未知的旅程，未来，一切都未可知，明天是否还有饭吃，都是个未知数，只是为了能在年轻的时候，多挣点钱，多存点钱，能有地方住，能看的起病。
    35岁，一个噩耗的年龄，无论是事业单位，还是私企，都不能容得下这些中年人的求职。听起来，感觉这个世界就已经抛弃了IT界35岁的人了。我也不得不承认，我不能保证35岁就能成为一名优秀的程序员。人生本来就是充满了苦难，生而为人，唯一能做的就是小心翼翼，减少犯错的机会，至少能选择一种让自己舒服的受罪的方式。明天的路该往何处去才能不让自己陷入万劫不复的深渊，难道真的要去卖山东杂粮煎饼吗？

    未雨绸缪，才能落于不败之地。在今后的很长时间里，我都应该不断的拷问自己，35岁危机，是否真的适用于自己身上。

    学习的本质，不在于记住哪些知识，而在于它触发了你的思考。—————————— 迈克尔·桑德尔

    人生要做很多次选择，在事业单位稳定却买不起房，难道我还是要一直住在宿舍，一直等着房价下降吗？等几年呢？五年？十年？还是十五年？房价下降的时候，国家经济也面临着崩溃，国民经济倒退，而在事业单位的我的工资不会少，这个时候，我便有了足够的能力和资金来购买属于我的房子了，这就是事业单位的强大的竞争力，与之相反的境地就是，在国家经济衰退的时候，私企会倒闭，会破产，对我来说，我就可能失去重要收入，甚至可能面临失业，对于巨大的房贷压力，也可能无力偿还，最终崩溃。我应该把所有的希望都压在国家上吗？

    2019年7月26日 星期五

    或许明天我会失业，也或许明天我会流落街头，但明天总会来临，充满希望的活下去，是我唯一能做的事情了。

    2019年7月22号 星期一

    明天是好还是坏？没有人知道，也没有人可以指导自己。

    2019年7月18日 星期四

    人生的重大选择，但愿能得到眷顾。人生的下一步和人生的上一步，哪个更好？如果你停留在原地，一切都是未知的。

    2019年7月17日 星期三

    什么叫虚度年华？每天996的工作，算是虚度年华吗？

    求其上者得其中，求其下着，而无所得。作为一个普通的不能再普通的人，该如何生存下去？
    城市，就是一个竞技场，高昂的房价，物价，成为了物竞天择的残酷条件，扔掉了那些没有能力的弱鸡，留下了可以一个月拿两万工资的精英人士。其实，也没有什么可以感叹了。

    世界上有无数平庸之人，而我，只是其中一个。

    这个世界本就是充满了灾难，人生而为受苦而来，无论你在何时在何地，命运中都背负着巨大的耻辱，生而有之，逃不掉，摆不脱。在这无尽的恶性循环之中，只有不断的麻痹自己，不断的用美好的未来这一个梦想让自己沉醉，心里才能得到片刻的安宁。谁说鸡汤没有用呢，谁说新闻联播没有用呢，谁说学习强国是一个洗脑工具呢，既然逃不掉，也只有沦为让人成功的垫脚石罢了，思想也只能任人摆布了，人的一生始终活在虚假繁荣的梦里，又何尝不是一种幸福。

    2019年02月01日 星期五

    一个人怎么样才算成熟了呢？18岁？不见得自己十八岁的时候就成熟了。十八岁，我生理年龄已经成熟了，可以肆意妄为然后被判入狱了,可以杀人放火而不用找监护人承担责任了，但是，我不得不承认，我的心理年龄却依然是个低能儿。
    2019年，我27岁，已经迈进了而立之年。我才发现，我依然不够成熟。回想自己大学时，高中时的经历和所做所为（很多我都记不起来了，我也不愿意去记忆了），很幼稚也很滑稽，有时候很想回去教育下自己，让自己能做的更好一点，奈何没有回头路可以走。

    很想忘记过去（我到底经历了什么，鬼知道呢），忘记自己的不成熟，忘记自己的疯狂，忘记自己的不堪回首的往事，从新开始。越想，却越逃不掉命运的惩罚，越挣扎，陷得也越深。每当看见过去的朋友，每当想起过去的点滴，我都心理忐忑，不知所措。我无法描述那种心情，哭不出来，也平静不下去。或许真的就像当初自己立下的誓言，愿意用一生的悲苦，偿还过去所欠下的罪孽，这剩下的半辈子啊，或许真的不会再幸福了。

    人，只有幸福了，才能平静的接受自己曾经的不幸。如今内心的每一次不安，或许都是源于自己如今生活的一塌糊涂。买车、买房、结婚、生孩子，该怎样逃脱命运的枷锁。没有人教我。如果我现在比任何一个人过得都幸福，也不会胡思乱想了。我在忏悔吗？我在后悔吗？我在责备自己吗？哪些是真，哪些又是假？我不再是最优秀的了，也不再是最温柔的了，更不是最值得依靠的了，什么都不剩的我，活该被抛弃，活该被世界遗忘。这都是命啊。

    越来越佩服钱学森的一生，励志报国，勤学苦读，终不负辛苦，成就大业。苦一生，成一世;大灾大难，成大荣大誉。而我，却依然在纠结着，挣扎着，却无法将心沉下。每个人都有每个人一生要走的路，苦也好，甜也罢，如果不能从过去的沼泽中汲取经验，越陷越深，等到只会是黑暗。

    2018年12月15日 星期六

    青春可是不等我们的呀。

    洛克 李，火影中的主要配角。是一个悲情的人物但也是一个值得学习与尊敬的人物。看似神经质和无厘头的义愤填膺，却包含着无数的汗水和泪水。相较于任何其他主人公来说，他不会幻术，没有血继极限，没有一个牛逼的老爹，也从没有外挂，在天才满地的木叶，高手林立的忍者世界，他的出现，看起来似乎有点悲凉，有着逗逼的眉毛和奇怪的发型，智商也有些不及常人，可以说，他一无是处。奇怪的是，他有梦想，那就是证明就算不会忍术，也能成为一个优秀的忍者。对，他只会体术。为了实现这个梦想，他比所有人都要来的努力，比鸣人，比佐助，比雏田，比小樱，比其他任何一个人，他的一天中，除了睡觉，所有的时间都在用来训练，他付出的努力，远远比其他人多的多的多，只是因为他什么天赋都没有，只是因为他什么背景都没有，只是因为他有一个普通的梦想。和他同样的人物，他的恩师迈特凯，对他的评价是：努力的天才

    2017年4月17日 星期一

    这个世界到底是什么样的，用我的双眼看不清，用我的双耳也听不明。社会上充斥着各种新闻报道，也渗透着我党伟大的光辉，国家富强人民安康伟大领袖，国家在实现中国梦的路上，越走越近。这就是我所听见的世界，这是新闻里的中国，却和我看见的经历的世界不是一个样子。我也不相信向外国反华人士对中国的片面的黑化，更不相信邪教组织对世界的扭曲，国家是漏洞百出，千疮百孔，人心浮躁，物欲横流，却还是一样的和平，即便这和平的掩盖下依然有看不见的动荡。
    我所见到的世界，是富人的世界。作为一个在杭漂泊的异乡之客，在错过了2015、2016、2017年的房价疯狂开涨的时期之后，即便有了二十万的积蓄，却也已经无力在杭州购买房子了。房价，支撑着这个国家的经济运行，国家GDP一半的来源都是房地产，国家不允许它崩盘，也不允许他下降，而我却到了该结婚该生孩子的年纪，只能独自哀叹，这就是赤裸裸的现实。房价所创造的繁荣世界里，只活着有钱人，有钱的人会越来越有钱，而没钱的人，一辈子都不会有钱。
    每当别人谈论房价的时候，我都不愿意出声，既然混不下去，为何还要在杭州苦苦挣扎，换一个城市是不是生活的会更好点呢？

    isNaN 和 Number.isNaN 函数的区别？

    1.函数 isNaN 接收参数后，会尝试将这个参数转换为数值，任何不能被转换为数值的的值都会返回 true，因此非数字值传入也会返回 true ，会影响 NaN 的判断。

    2.函数 Number.isNaN 会首先判断传入参数是否为数字，如果是数字再继续判断是否为 NaN ，不会进行数据类型的转换，这种方法对于 NaN 的判断更为准确。

    为什么0.1+0.2 ! == 0.3，如何让其相等?

    一个直接的解决方法就是设置一个误差范围，通常称为“机器精度”。对JavaScript来说，这个值通常为2-52，在ES6中，提供了Number.EPSILON属性，而它的值就是2-52，只要判断0.1+0.2-0.3是否小于Number.EPSILON，如果小于，就可以判断为0.1+0.2 ===0.3。

    == 操作符的强制类型转换规则？

    1.首先会判断两者类型是否**相同，**相同的话就比较两者的大小。

    2.类型不相同的话，就会进行类型转换。

    3.会先判断是否在对比 null 和 undefined，是的话就会返回 true。

    4.判断两者类型是否为 string 和 number，是的话就会将字符串转换为 number。

    5.判断其中一方是否为 boolean，是的话就会把 boolean 转为 number 再进行判断。

    6.判断其中一方是否为 object 且另一方为 string、number 或者 symbol，是的话就会把 object 转为原始类型再进行判断。
