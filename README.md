#这是第一个用hexo和github搭建的博客系统

  - 技术路线 hexo+git+github
  
  - 系统启动说明
    - 安装 Node.js
      - windows使用nvm
      - nvm install 10.16.2
    - 安装 Git
    - 安装 Hexo
        npm install -g hexo-cli
    - 初始化hexo目录
        hexo init
    - 进入hexo目录执行 npm install
    - hexo 常用命令
      - hexo server 或者 hexo s启动hexo本地服务器
      - hexo generate
